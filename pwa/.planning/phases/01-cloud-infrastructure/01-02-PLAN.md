**Step 2: UFW Firewall Setup**
Run these commands in exact order (allow SSH FIRST to prevent lockout):
```bash
sudo ufw allow 22/tcp
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw default deny incoming
sudo ufw default allow outgoing
sudo ufw enable
sudo ufw status verbose
```

**Step 3: Swap Setup**
```bash
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
free -h
```

**Step 4: Deploy Directory Setup**
```bash
# Create /opt/racingpoint/ directory structure
sudo mkdir -p /opt/racingpoint
cd /opt/racingpoint

# Clone or pull racecontrol repo
git clone https://github.com/james-racingpoint/racecontrol.git 2>/dev/null || (cd racecontrol && git pull)

# Clone or pull admin repo
git clone https://github.com/james-racingpoint/racingpoint-admin.git 2>/dev/null || (cd racingpoint-admin && git pull)

# Copy cloud configs to deploy directory
cp racecontrol/cloud/Caddyfile ./Caddyfile
cp racecontrol/cloud/compose.yml ./compose.yml
```

**Step 5: Docker Compose Up**
```bash
cd /opt/racingpoint
docker compose up -d --build
docker compose ps
docker compose logs caddy --tail=20
```
