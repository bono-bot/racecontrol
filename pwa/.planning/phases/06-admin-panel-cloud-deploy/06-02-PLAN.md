```bash
# Expected: Let's Encrypt issuer

# 6. Existing PWA still works (no regression)
curl -sI https://app.racingpoint.cloud | head -3
# Expected: HTTP/2 200
```

If step 4 returns "rc-core unreachable", that is acceptable — same known issue as Phase 2 (racecontrol binary may not be running on VPS host). The admin container itself is working correctly.
