  - Below boxes: character grid. Layout: 7 columns x 5 rows. Characters from `ABCDEFGHJKMNPQRSTUVWXYZ23456789` in order, plus Backspace and Clear buttons.
  - Each grid button: ~56px square, `bg-[#222222] hover:bg-[#333333] text-white font-semibold rounded-lg` border border-[#333333].
  - Bottom row: "Clear" button (spans 2 cols), "Submit" button (spans 3 cols, `bg-[#E10600]` Racing Red, disabled if pin.length != 6).

  - Auto-focus behavior: no text input to focus — grid buttons handle input.

  **Validating State (`validating`):**
  - Show the 6 filled boxes (frozen)
  - Spinner/loading indicator below
  - "Validating PIN..." text

  **Success State (`success`):**
  - Large pod number display: "Head to Pod X" with X being the pod_number in very large text (`text-8xl font-bold text-[#E10600]`).
  - Below: experience_name, tier_name, formatted allocated time (e.g., "30 minutes").
  - "Your game is loading..." with a subtle animated pulse or spinner.
  - Auto-close after 15 seconds (return to landing page via onClose).

  **Error State (`error`):**
  - "Invalid PIN" in red text.
  - Show `remaining_attempts` if present: "X attempts remaining" in yellow/amber.
  - If the error contains "being processed", show a friendlier message with a clock icon.
  - "Try Again" button that resets to entry state (clears pin).
  - Auto-return to entry state after 10 seconds.

  **Lockout State (`lockout`):**
  - "Too Many Attempts" heading.
  - Live countdown timer showing minutes:seconds remaining (derive from `lockout_remaining_seconds`, tick every second with setInterval).
  - "Please wait" message.
  - When timer reaches 0, auto-transition to entry state.

  **Handler logic:**
  ```typescript
  async function handleSubmit() {
    if (pin.length !== 6) return;
    setStep("validating");
    try {
      const res = await api.redeemPin(pin);
      if (res.lockout_remaining_seconds) {
        setLockoutSeconds(res.lockout_remaining_seconds);
        setStep("lockout");
      } else if (res.error) {
        setErrorMsg(res.error);
        setRemainingAttempts(res.remaining_attempts ?? null);
        setStep("error");
      } else {
        setResultPodNumber(res.pod_number ?? 0);
        setResultExperience(res.experience_name ?? "");
        setResultTier(res.tier_name ?? "");
        setResultSeconds(res.allocated_seconds ?? 0);
        setStep("success");
      }
    } catch {
      setErrorMsg("Network error - please try again");
      setStep("error");
    }
  }
  ```
