    Public function: `pub fn scan_ac_content() -> ContentManifest` -- the main entry point.
    Also expose `pub fn scan_ac_content_at(content_path: &Path) -> ContentManifest` for testability (takes arbitrary path instead of hardcoded AC path).

    Internal functions:
    - `fn scan_cars(cars_dir: &Path) -> Vec<CarManifestEntry>` -- read_dir, filter directories, skip hidden, collect IDs
    - `fn scan_tracks(tracks_dir: &Path) -> Vec<TrackManifestEntry>` -- for each track folder, detect configs

    - Pitfall 3: pitboxes is a STRING in ui_track.json, not an integer
    - Pitfall 4: ui_track.json location varies -- default: ui/ui_track.json, named config: ui/{config}/ui_track.json
    - Pitfall 6: Empty ai/ folder -- check read_dir count > 0, not just directory existence
