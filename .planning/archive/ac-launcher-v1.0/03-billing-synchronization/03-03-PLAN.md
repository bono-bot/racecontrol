<objective>
Wire the rc-core side of billing synchronization: WebSocket handler dispatches GameStatusUpdate to billing functions, auth is decoupled from immediate billing start, and launch timeout handling ensures 3-min timeout with retry and cancel-on-double-failure.

Purpose: BILL-01 requires billing to start only on STATUS=LIVE, not at PIN validation. This plan implements the core-side logic that receives GameStatusUpdate from the agent (Plan 02 sends it), triggers billing start/pause/resume accordingly, and handles the game launch failure scenario (3-min timeout, auto-retry once, cancel on second fail with no charge). Without this plan, the agent sends GameStatusUpdate messages but core ignores them.

Output: WebSocket handler match arm for GameStatusUpdate, billing lifecycle functions (start on Live, pause on Pause, resume on re-Live), decoupled auth/mod.rs, and launch timeout tracking with retry/cancel logic.
</objective>

  ```rust
  pub async fn handle_game_status_update(
      state: &Arc<AppState>,
      pod_id: &str,
      ac_status: AcStatus,
      cmd_tx: &tokio::sync::mpsc::Sender<CoreToAgentMessage>,
  ) {
      match ac_status {
          AcStatus::Live => {
              // Check if this pod is in waiting_for_game -- if so, start billing
              let entry = state.billing.waiting_for_game.write().await.remove(pod_id);
              if let Some(entry) = entry {
                  // Start billing now that AC is LIVE
                  match start_billing_session(
                      state,
                      entry.pod_id,
                      entry.driver_id,
                      entry.pricing_tier_id,
                      entry.custom_price_paise,
                      entry.custom_duration_minutes,
                      entry.staff_id,
                      entry.split_count,
                      entry.split_duration_minutes,
                  ).await {
                      Ok(session_id) => {
                          tracing::info!("Billing started on LIVE for pod {} (session {})", pod_id, session_id);
                      }
                      Err(e) => {
                          tracing::error!("Failed to start billing on LIVE for pod {}: {}", pod_id, e);
                      }
                  }
              } else {
                  // No waiting entry -- check if timer exists and is PausedGamePause (resume)
                  let mut timers = state.billing.active_timers.write().await;
                  if let Some(timer) = timers.get_mut(pod_id) {
                      if timer.status == BillingSessionStatus::PausedGamePause {
                          timer.status = BillingSessionStatus::Active;
                          timer.pause_seconds = 0; // reset pause counter on resume
                          tracing::info!("Billing resumed on LIVE for pod {} (was PausedGamePause)", pod_id);
                      }
                      // If already Active, this is a no-op (idempotent)
                  }
              }
          }
          AcStatus::Pause => {
              let mut timers = state.billing.active_timers.write().await;
              if let Some(timer) = timers.get_mut(pod_id) {
                  if timer.status == BillingSessionStatus::Active {
                      timer.status = BillingSessionStatus::PausedGamePause;
                      timer.pause_seconds = 0; // start counting pause duration from 0
                      timer.pause_count += 1;
                      tracing::info!("Billing paused (game pause) for pod {}", pod_id);
                  }
              }
              // If no active timer, Pause is a no-op
          }
          AcStatus::Off => {
              // Game exited -- if there's an active billing timer, end the session
              let has_timer = {
                  let timers = state.billing.active_timers.read().await;
                  timers.contains_key(pod_id)
              };
              if has_timer {
                  tracing::info!("Game exited (STATUS=Off) for pod {}, ending billing session", pod_id);
                  end_billing_session(state, pod_id, "game_exit").await;
              }
              // Also remove from waiting_for_game if present (game crashed during loading)
              state.billing.waiting_for_game.write().await.remove(pod_id);
          }
          AcStatus::Replay => {
              // Replay mode -- treat same as Pause for billing purposes
              // (customer watching a replay is not actively driving)
              let mut timers = state.billing.active_timers.write().await;
              if let Some(timer) = timers.get_mut(pod_id) {
                  if timer.status == BillingSessionStatus::Active {
                      timer.status = BillingSessionStatus::PausedGamePause;
                      timer.pause_seconds = 0;
                      timer.pause_count += 1;
                      tracing::info!("Billing paused (replay) for pod {}", pod_id);
                  }
              }
          }
      }
  }
  ```

  pub async fn check_launch_timeouts(state: &Arc<AppState>) -> Vec<(String, u8)> {
      // Returns list of (pod_id, attempt) for pods that have timed out
      let mut timed_out = Vec::new();
      let waiting = state.billing.waiting_for_game.read().await;
      for (pod_id, entry) in waiting.iter() {
          if entry.waiting_since.elapsed() > Duration::from_secs(180) {
              timed_out.push((pod_id.clone(), entry.attempt));
          }
      }
      timed_out
  }
