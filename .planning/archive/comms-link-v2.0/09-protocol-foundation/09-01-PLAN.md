2. `AckTracker` class extends EventEmitter:
   - Constructor: `{ sendFn, nowFn = Date.now, timeoutMs = 10000, maxRetries = 3 }`
   - `#pending` Map: messageId -> { rawMessage, seq, attempt, timer }
   - `#seq = 0`: monotonic counter
   - `track(messageId, rawMessage, messageType)`: if isControlMessage(messageType) throw Error('Cannot track control message'). Assign seq = this.#seq++. Call sendFn(rawMessage). Set retry timer via setTimeout. Store in pending map. Return seq.
   - `acknowledge(messageId)`: if pending.has(messageId), clearTimeout(entry.timer), delete from pending, emit('ack', messageId)
   - Private `#scheduleRetry(messageId)`: setTimeout at `timeoutMs * 2 ** entry.attempt`. On fire: if attempt < maxRetries, increment attempt, call sendFn again, emit('retry', {messageId, attempt}), schedule next retry. If attempt >= maxRetries, emit('timeout', messageId), delete from pending.
   - `getPendingMessages()`: Array.from(pending.values()).sort((a,b) => a.seq - b.seq).map(e => e.rawMessage)
   - `reset()`: for each pending entry clearTimeout(timer), then clear map
   - `get pendingCount()`: pending.size
   - `get currentSeq()`: #seq
