GREEN phase: Create shared/message-queue.js:

```javascript
import { EventEmitter } from 'node:events';

export class MessageQueue extends EventEmitter {
  #entries = [];             // Array of { id, ts, type, payload, acked }
  #walPath;
  #appendFileFn;
  #readFileFn;
  #writeFileFn;
  #nowFn;
  #maxSize;

  constructor({ storePath, appendFileFn, readFileFn, writeFileFn, nowFn = Date.now, maxSize = 1000 }) {
    super();
    this.#walPath = storePath;
    this.#appendFileFn = appendFileFn;
    this.#readFileFn = readFileFn;
    this.#writeFileFn = writeFileFn;
    this.#nowFn = nowFn;
    this.#maxSize = maxSize;
  }
```

Implementation details:

1. `async load()`:
   - Try readFileFn(walPath, 'utf8'). Catch ENOENT -> return (empty queue).
   - Split by '\n', filter empty lines.
   - For each line: try JSON.parse. On fail (partial line from crash), skip it.
   - Build entries array. Then process ACK lines: any line with only `{id, acked:true}` marks the matching entry as acked.
   - Filter out acked entries from #entries (they're resolved).

2. `async enqueue(message)`:
   - message must be an object with at least `id` field.
   - If #entries.filter(e => !e.acked).length >= maxSize, throw Error('Queue at capacity').
   - Create entry: `{ id: message.id, ts: this.#nowFn(), type: message.type, payload: message.payload ?? {}, acked: false }`
   - Append to WAL: `await this.#appendFileFn(this.#walPath, JSON.stringify(entry) + '\n')`
   - Push to #entries.
   - Emit 'enqueue' with message.
   - Return message.id.

3. `acknowledge(messageId)`:
   - Find entry in #entries by id. If not found, return (no-op).
   - Set entry.acked = true.
   - Append ACK line to WAL: `await this.#appendFileFn(this.#walPath, JSON.stringify({id: messageId, acked: true}) + '\n')`
   - Emit 'ack' with messageId.

4. `async compact()`:
   - Filter #entries to unacked only.
   - Rewrite WAL via writeFileFn: each unacked entry as JSON line.
   - Count removed = old length - new length.
   - Update #entries to only unacked.
   - Emit 'compact' with removed count.

5. `getPending()`: return #entries.filter(e => !e.acked).map(e => ({ id: e.id, ts: e.ts, type: e.type, payload: e.payload }))

6. `get size()`: return #entries.filter(e => !e.acked).length
