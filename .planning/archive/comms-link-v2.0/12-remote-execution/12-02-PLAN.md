```javascript
constructor({
  execFileFn,           // (binary, args, opts, cb) => void -- defaults to node:child_process execFile
  sendResultFn,         // (execId, result) => void -- REQUIRED
  notifyFn,             // (text) => Promise -- WhatsApp notification
  nowFn = Date.now,     // Injectable clock
  approvalTimeoutMs = 600000,  // 10 minutes default
  commandRegistry = COMMAND_REGISTRY,
  safeEnv = buildSafeEnv(),
})
```

Methods:
- **handleExecRequest(msg)** -- msg has { payload: { execId, command, reason, requestedBy } }
  1. Check completedExecs Set for dedup -- if seen, return silently
  2. Validate command via validateExecRequest -- if unknown, send rejection result
  3. Route by tier: AUTO -> execute, NOTIFY -> execute + notify, APPROVE -> queueForApproval

- **#execute(execId, command, spec)** -- private
  1. Emit 'exec_started'
  2. Call execFileFn with { env: safeEnv, timeout: spec.timeoutMs, maxBuffer: 5*1024*1024, cwd: spec.cwd || undefined, shell: false }
  3. On callback: build result { execId, command, exitCode, stdout (truncated at 50000), stderr (truncated at 10000), durationMs, truncated (bool), tier }
  4. Add execId to completedExecs
  5. Call sendResultFn
  6. Emit 'exec_completed'

- **#queueForApproval(execId, command, spec, requestedBy)** -- private
  1. Set timeout (approvalTimeoutMs) -- on timeout: delete from pending, send timed_out result, emit 'approval_timeout'
  2. Add to pendingApprovals Map: { command, spec, requestedBy, timer }
  3. Emit 'pending_approval'
  4. Call notifyFn with approval URL containing execId

- **approveCommand(execId)** -- returns boolean
  1. Get from pendingApprovals -- if not found, return false
  2. Clear timer, delete from pending
  3. Execute the command
  4. Return true

- **rejectCommand(execId, reason)** -- returns boolean
  1. Get from pendingApprovals -- if not found, return false
  2. Clear timer, delete from pending
  3. Send rejection result via sendResultFn
  4. Return true

- **get pendingApprovals** -- returns array of { execId, command, requestedBy }

- **shutdown()** -- clears all timers, clears pendingApprovals map

CRITICAL: execFile call MUST include `shell: false` explicitly. MUST include `env: this.#safeEnv`.
