    - #reconnectAttempt = 0
    - #intentionalClose = false
    - #queue = []
    - #maxQueueSize (from constructor, default 100)

    Add constants at module level:
    - BACKOFF_BASE = 1000 (1 second)
    - BACKOFF_MAX = 30_000 (30 seconds)
    - JITTER_MAX = 500 (0-500ms)

    Modify constructor: Accept optional maxQueueSize parameter, default to 100.

    Add getter: get queueSize() { return this.#queue.length; }

    Modify connect():
    - Add `clearTimeout(this.#reconnectTimer); this.#reconnectTimer = null;` at the top (before cleanup).

    Modify open handler (inside connect()):
    - Reset: this.#reconnectAttempt = 0
    - Clear: this.#intentionalClose = false
    - Transition: this.sm.transition(State.CONNECTED)
    - Flush: this.#flushQueue()   <-- BEFORE emit('open') to prevent interleaving
    - Emit: this.emit('open')

    Modify close handler (inside connect()):
    - If this.#intentionalClose: transition to DISCONNECTED (guard against already DISCONNECTED), emit close, return.
    - Else: if not already RECONNECTING, transition to RECONNECTING. Emit close. Call this.#scheduleReconnect().

    Modify send():
    - If CONNECTED and ws.readyState === WebSocket.OPEN: ws.send(msg), return true.
    - Else: if queue full (>= maxQueueSize), shift oldest. Push msg. Return false.

    Modify disconnect():
    - Set this.#intentionalClose = true
    - clearTimeout(this.#reconnectTimer); this.#reconnectTimer = null

    - clearTimeout(this.#reconnectTimer)
    - Calculate delay: Math.min(BACKOFF_BASE * Math.pow(2, this.#reconnectAttempt), BACKOFF_MAX)
