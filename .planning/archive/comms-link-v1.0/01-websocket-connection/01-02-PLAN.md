<task type="auto" tdd="true">
  <name>Task 1: Bono server + James client implementation</name>
  <files>bono/comms-server.js, bono/index.js, james/comms-client.js, james/index.js</files>
  <behavior>
    - Server rejects connection without Authorization header (401)
    - Server rejects connection with wrong PSK (401)
    - Server accepts connection with correct PSK
    - Server echoes back echo messages as echo_reply
    - Server pings connected clients every 25 seconds (transport-level keepalive)
    - Server terminates clients that don't respond to ping within 25 seconds
    - Client sends PSK in Authorization: Bearer header during upgrade
    - Client transitions state machine to CONNECTED on open
    - Client transitions state machine to DISCONNECTED on close
    - Client.send(type, payload) returns true when connected, false when disconnected
    - Client emits 'message' event when receiving parsed messages from server
    - Client cleans up old WebSocket before creating new one on connect()
  </behavior>
  <action>
    1. Create `bono/comms-server.js`:
       - Export `createCommsServer({ port, psk })` function that returns `{ server, wss, start(), stop() }`
       - Use `http.createServer()` + `new WebSocketServer({ noServer: true })`
       - In `server.on('upgrade')` handler:
         - Register error handler on socket
         - Extract token from `Authorization: Bearer <token>` header
         - If no token or invalid PSK, write `HTTP/1.1 401 Unauthorized\r\n\r\n`, destroy socket, return
         - PSK validation: use `crypto.timingSafeEqual()` with Buffer.from() on both sides. Check length first (different lengths = reject without timing leak)
         - On valid PSK: remove error handler, call `wss.handleUpgrade`, emit connection
       - In `wss.on('connection')` handler:
         - Set `ws.isAlive = true`
         - Register `ws.on('pong', () => { ws.isAlive = true; })`
         - Register `ws.on('message')` handler:
           - Parse with `parseMessage(data.toString())`
           - If type === 'echo', respond with `createMessage('echo_reply', 'bono', msg.payload)`
           - Emit `'message'` event on wss with `(msg, ws)` for future extensibility
         - Register `ws.on('error', () => {})` to prevent unhandled error crash
       - Set up ping interval (25 seconds):
         - For each client in `wss.clients`: if `ws.isAlive === false`, terminate. Otherwise set `ws.isAlive = false` and send `ws.ping()`
         - Clear interval on `wss.on('close')`
       - `start()`: calls `server.listen(port)`, returns a promise that resolves when listening
       - `stop()`: closes wss, closes server, clears ping interval, returns promise

    2. Create `bono/index.js`:
       - Import createCommsServer
       - Read PSK from `process.env.COMMS_PSK` -- if missing, log error and exit with code 1
       - Read port from `process.env.COMMS_PORT` or default to 8765
       - Call `createCommsServer({ port, psk })`
       - Call `start()`
       - Log `'Comms server listening on port {port}'`
       - Handle SIGTERM/SIGINT gracefully: call stop(), then process.exit(0)

    3. Create `james/comms-client.js`:
       - Import WebSocket from 'ws'
       - Import { createMessage, parseMessage } from '../shared/protocol.js'
       - Import { ConnectionStateMachine, State } from '../shared/state.js'
       - Export `CommsClient` class:
         - Constructor takes `{ url, psk }`. Store them. Create `this.sm = new ConnectionStateMachine()`. Forward sm 'state' events: `this.sm.on('state', (evt) => this.emit('state', evt))`
         - CommsClient extends EventEmitter (import from node:events)
         - Private `#ws = null`
         - Getter `state` returns `this.sm.state`
         - Method `connect()`:
           - If `#ws` exists: call `#ws.removeAllListeners()`, call `#ws.terminate()`, set `#ws = null`
           - Create `new WebSocket(url, { headers: { 'Authorization': 'Bearer ' + psk } })`
           - On 'open': transition state machine to CONNECTED, emit 'open'
           - On 'close': transition state machine to DISCONNECTED (only if current state !== DISCONNECTED, to avoid invalid same-state transition), emit 'close'
           - On 'message': parse with parseMessage(data.toString()), emit 'message' with parsed msg
           - On 'error': emit 'error' with error object (do NOT transition state here -- close event follows)
         - Method `send(type, payload)`:
           - If state !== CONNECTED, return false
           - `#ws.send(createMessage(type, 'james', payload))`
           - Return true
         - Method `disconnect()`:
           - If `#ws`: `#ws.close(1000, 'client disconnect')`, `#ws = null`

    4. Create `james/index.js`:
       - Import CommsClient
       - Read PSK from `process.env.COMMS_PSK` -- if missing, log error and exit with code 1
       - Read server URL from `process.env.COMMS_URL` or default to `ws://localhost:8765`
       - Create client instance, call connect()
       - Log state changes: `client.on('state', (evt) => console.log('State:', evt.state))`
       - On 'open': send an echo message, log it
       - On 'message': log received message
       - Handle SIGTERM/SIGINT: call disconnect(), exit

    NOTE on PSK in headers (not query params) per research recommendation: `headers: { 'Authorization': 'Bearer ' + psk }` -- headers don't leak into server logs. Do NOT use query parameters.
