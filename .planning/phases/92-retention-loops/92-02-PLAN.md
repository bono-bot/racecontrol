**Step 2 — Compute streak urgency helper:**

Add a helper function or inline computation near the top of the component (inside the component function, after data fetch):

```typescript
// Streak urgency: compute days until grace expires
const daysUntilGraceExpires = (() => {
  if (!summary?.grace_expires_date) return null;
  const expires = new Date(summary.grace_expires_date + 'T23:59:59+05:30');
  const now = new Date();
  const diffMs = expires.getTime() - now.getTime();
  const diffDays = Math.ceil(diffMs / (1000 * 60 * 60 * 24));
  return diffDays > 0 ? diffDays : 0;
})();
const streakAtRisk = daysUntilGraceExpires !== null && daysUntilGraceExpires <= 7 && (summary?.streak_weeks ?? 0) >= 1;
```

**Step 3 — Replace the existing streak card (~line 189-194):**

Replace the single "Week Streak" card with an enhanced version:

```tsx
<div className={`bg-rp-card border rounded-xl p-4 ${streakAtRisk ? 'border-[#E10600]' : 'border-rp-border'}`}>
  <p className="text-xs text-rp-grey mb-1">Week Streak</p>
  <p className="text-2xl font-bold text-white">
    {summary.streak_weeks}
  </p>
  {streakAtRisk && daysUntilGraceExpires !== null && (
    <p className="text-xs text-[#E10600] mt-1 font-medium">
      {daysUntilGraceExpires === 0
        ? 'Expires today!'
        : `${daysUntilGraceExpires}d left`}
    </p>
  )}
  {(summary.longest_streak ?? 0) > (summary.streak_weeks ?? 0) && (
    <p className="text-xs text-rp-grey mt-1">
      Best: {summary.longest_streak}w
    </p>
  )}
</div>
```

- Show days remaining countdown in red text below the streak number

- No animations (PWA is lightweight)
- Uses existing Tailwind classes (`bg-rp-card`, `text-rp-grey`) plus inline Racing Red for urgency

**Step 4 — Verify no hydration issues:**

The `daysUntilGraceExpires` computation uses `new Date()` which differs server/client. Since this page likely already uses `"use client"` directive (it fetches data), this is fine. If not, wrap the urgency computation in a `useEffect` + state pattern:

const [daysLeft, setDaysLeft] = useState<number | null>(null);

  if (!summary?.grace_expires_date) return;
  const expires = new Date(summary.grace_expires_date + 'T23:59:59+05:30');
  const diffMs = expires.getTime() - Date.now();
  setDaysLeft(Math.max(0, Math.ceil(diffMs / (1000 * 60 * 60 * 24))));
