**Function 2: `maybe_grant_variable_reward` (RET-03, RET-06)**

```rust
pub async fn maybe_grant_variable_reward(
    state: &Arc<AppState>,
    driver_id: &str,
    trigger: &str,
) {
```
- Probability gate: `pb` = 0.15, `milestone` = 0.10, anything else returns early
- Use `rand::thread_rng().gen_bool(threshold)` — return if false
- Compute IST month string: `chrono::FixedOffset::east_opt(5 * 3600 + 30 * 60).unwrap()` then `now_ist.format("%Y-%m").to_string()`
- Query `wallets.total_debited_paise` for driver; compute `cap_paise = total_spend / 20` (5%)
- Query `SELECT COALESCE(SUM(amount_paise), 0) FROM variable_reward_log WHERE driver_id = ? AND month = ?`
- If `already_rewarded >= cap_paise`, log and return
- Base amount: pb = 5000 paise (50 credits), milestone = 10000 paise (100 credits)
- Clamp: `amount = base_amount.min(cap_paise - already_rewarded)`, return if <= 0
- Generate UUID for reward_id
- Call `crate::wallet::credit(state, driver_id, amount, "bonus", Some(&reward_id), Some(&format!("Surprise bonus — {}", trigger)), None).await`
- INSERT into `variable_reward_log (id, driver_id, amount_paise, trigger, month, created_at)` with the IST month string
- Log with `tracing::info!`
