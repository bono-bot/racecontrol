Create `deploy/pod-lockdown.ps1` — a one-time idempotent pod setup script that prevents system UI from leaking to the customer display.

Per user decisions:
- Taskbar: Hide via registry (not just auto-hide — fully hidden). Requires Explorer restart.
- Win key: Block via registry NoWinKeys.
- Windows Update: Suppress restart notifications via Group Policy registry keys.
- Keyboard shortcuts: Block Win key and Ctrl+Esc. Accept Alt+Tab (per research recommendation — customer only sees lock screen behind game).
- Recovery: Admin login required to undo. Changes are idempotent and safe to re-apply.
- These are one-time pod setup changes, NOT rc-agent runtime code.

Script contents:

```powershell
# pod-lockdown.ps1 — One-time kiosk lockdown for Racing Point pods
# Run as Administrator on each pod (via pod-agent /exec or manual install)
# Idempotent: safe to re-run. Requires Explorer restart (automatic).
#
# Reverts: To undo, delete the registry keys and restart Explorer.
# Admin login is required to access regedit/powershell on locked-down pods.

param(
    [switch]$Undo  # Pass -Undo to revert all lockdown changes
)

$ErrorActionPreference = 'SilentlyContinue'

if ($Undo) {
    Write-Host "[pod-lockdown] Reverting lockdown changes..."

    # Restore taskbar visibility
    $p = 'HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\StuckRects3'
    $v = (Get-ItemProperty -Path $p).Settings
    if ($v) { $v[8] = 2; Set-ItemProperty -Path $p -Name Settings -Value $v }  # 2 = visible

    # Remove Win key block
    Remove-ItemProperty -Path 'HKCU:\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer' -Name 'NoWinKeys' -ErrorAction SilentlyContinue

    # Remove Windows Update suppression
    Remove-ItemProperty -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU' -Name 'NoAutoRebootWithLoggedOnUsers' -ErrorAction SilentlyContinue
    Remove-ItemProperty -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU' -Name 'AUOptions' -ErrorAction SilentlyContinue

    Stop-Process -Name explorer -Force -ErrorAction SilentlyContinue
    Write-Host "[pod-lockdown] Lockdown reverted. Explorer restarting..."
    exit 0
}

Write-Host "[pod-lockdown] Applying kiosk lockdown..."

# 1. Auto-hide taskbar (customer cannot find it)
$p = 'HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\StuckRects3'
$v = (Get-ItemProperty -Path $p -ErrorAction SilentlyContinue).Settings
if ($v) {
    $v[8] = 3  # 3 = always-on-top + auto-hide (effectively hidden for kiosk)
    Set-ItemProperty -Path $p -Name Settings -Value $v
    Write-Host "[pod-lockdown] Taskbar auto-hide enabled"
} else {
    Write-Host "[pod-lockdown] WARNING: StuckRects3 not found — taskbar unchanged"
}

# 2. Block Win key (prevents Start Menu access)
$ep = 'HKCU:\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer'
New-Item -Path $ep -Force -ErrorAction SilentlyContinue | Out-Null
Set-ItemProperty -Path $ep -Name 'NoWinKeys' -Value 1 -Type DWord
Write-Host "[pod-lockdown] Win key blocked"

# 3. Suppress Windows Update restart notifications
$wup = 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU'
New-Item -Path $wup -Force -ErrorAction SilentlyContinue | Out-Null
Set-ItemProperty -Path $wup -Name 'NoAutoRebootWithLoggedOnUsers' -Value 1 -Type DWord
Set-ItemProperty -Path $wup -Name 'AUOptions' -Value 2 -Type DWord  # Download only, no auto-install
Write-Host "[pod-lockdown] Windows Update restart prompts suppressed"

# 4. Restart Explorer to apply taskbar change
Stop-Process -Name explorer -Force -ErrorAction SilentlyContinue
Write-Host "[pod-lockdown] Explorer restarted. Lockdown applied."
Write-Host ""
Write-Host "To revert: powershell -ExecutionPolicy Bypass -File pod-lockdown.ps1 -Undo"
```

This script:
- Is idempotent (safe to re-run on any pod)
- Has an -Undo flag for admin recovery
- Restarts Explorer to apply taskbar changes immediately
- Does NOT block Alt+Tab (per research: customer sees lock screen behind game, which is acceptable)
- Can be deployed via `pod-agent /exec` using a PowerShell one-liner that downloads and runs it, or manually via pendrive
