     - Loading state: animate-pulse text.
     - Error state: red text with retry button.
     - Camera grid: `grid grid-cols-1 lg:grid-cols-2 gap-4`.

       - Set `alt={camera.name}`, `className="w-full aspect-video object-cover bg-black"`.
       - If camera status is "disconnected" or "offline", show a dark overlay with "Camera Offline" text instead of the img tag.
     - Empty state if no cameras configured.
