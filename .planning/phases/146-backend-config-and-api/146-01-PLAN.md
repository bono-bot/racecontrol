Add the default function for zone:

```rust
fn default_zone() -> String {
    "other".to_string()
}
```

Add a helper method to CameraConfig (after the existing relay_url method) that returns the effective display name:

```rust
pub fn effective_display_name(&self) -> &str {
    self.display_name.as_deref().unwrap_or(&self.name)
}
```

All three fields use `#[serde(default)]` so existing TOML configs without these fields will still parse. display_name defaults to None (caller uses name as fallback), display_order defaults to None (treated as 0 by frontend), zone defaults to "other".
