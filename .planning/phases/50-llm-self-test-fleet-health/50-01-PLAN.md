    - `probe_ws_connected(status: &Arc<HeartbeatStatus>)` — read `status.ws_connected.load(Ordering::Relaxed)`
    - `probe_tcp_port(name, addr)` — use `tokio::task::spawn_blocking` + `TcpStream::connect_timeout(addr.parse().unwrap(), Duration::from_secs(1))`. Probes: lock_screen (127.0.0.1:18923), remote_ops (127.0.0.1:8090), overlay (127.0.0.1:18925), debug_server (127.0.0.1:18924)
    - `probe_udp_port(port)` — use `spawn_blocking` + `netstat -ano` output parsing. Search for `":{port} "` or `":{port}\r"` in lines containing "UDP". Probe 5 ports: 9996 (AC), 20777 (F1), 5300 (Forza), 6789 (iRacing), 5555 (LMU). Use a helper `probe_udp_port_from_netstat_output(port, netstat_stdout)` for testability.
    - `probe_hid()` — use `spawn_blocking` + `hidapi::HidApi::new()` then `api.device_list().any(|d| d.vendor_id() == 0x1209 && d.product_id() == 0xFFB0)`. CRITICAL: enumerate only, do NOT open the device (Pitfall 1 from research).
    - `probe_ollama(ollama_url)` — `reqwest::get(format!("{}/api/tags", ollama_url))`, check response contains "rp-debug" in model names. Use OLLAMA_CLIENT from self_monitor.rs or create a new client with 5s timeout.
    - `probe_close_wait()` — reuse same `netstat -ano` pattern as `count_close_wait_on_8090()` in self_monitor.rs. Pass if count < 20.
    - `probe_single_instance()` — `spawn_blocking` + `tasklist /FI "IMAGENAME eq rc-agent.exe"`. Count lines containing "rc-agent.exe". Pass if exactly 1.
    - `probe_disk()` — `spawn_blocking` + `sysinfo::Disks::new_with_refreshed_list()`, find C: drive, pass if available_space > 2_000_000_000 (2GB).
    - `probe_memory()` — `spawn_blocking` + `sysinfo::System::new()` + `system.refresh_memory()`, pass if `system.available_memory() > 1_073_741_824` (1GB).
    - `probe_shader_cache()` — `spawn_blocking` + check if `C:\Users\Public\AppData\Local\NVIDIA\GLCache` exists. Skip if dir not found, Pass if exists.
    - `probe_build_id()` — check `env!("GIT_HASH")` or use `option_env!("GIT_HASH")`. Pass if non-empty, Fail otherwise. Use `option_env!` to avoid compile failure if not set; return Skip with "GIT_HASH not set at compile time" if None.
    - `probe_billing_state(status: &Arc<HeartbeatStatus>)` — read `billing_active` atomic. Always Pass (billing state is informational). Detail: "billing_active={true|false}".
    - `probe_session_id()` — Skip with detail "session_id probe requires billing context". (Session ID is managed by billing_guard and not easily accessible from self_test; mark as Skip for now — the billing_state probe covers the critical path.)
    - `probe_gpu_temp()` — `spawn_blocking` + run `nvidia-smi --query-gpu=temperature.gpu --format=csv,noheader`. Parse stdout as integer. Pass if < 90, Fail if >= 90. Skip if nvidia-smi not found or command fails.
    - `probe_steam()` — `spawn_blocking` + `tasklist /FI "IMAGENAME eq steam.exe"`. Pass if found, Skip if not found (detail: "Steam not running — may be OK if no Steam games configured").
