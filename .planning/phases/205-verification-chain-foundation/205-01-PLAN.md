**ColdVerificationChain** — synchronous builder:
- `ColdVerificationChain::new(name: &str)` — creates chain with a name for the parent tracing span
- `.step(impl VerifyStep<Input=CurrentOutput, Output=NextOutput>)` — type-erased step via boxed closures internally. Since Rust doesn't support variadic generics, use a simpler design: chain stores a `Vec<Box<dyn FnOnce(...) -> Result<..., VerificationError>>>` approach is too complex. Instead, provide a `run_chain()` free function or a macro-less sequential approach:
  - Simplest viable design: `ColdVerificationChain` holds the chain `name: String` and provides `fn execute_step<S: VerifyStep>(&self, step: &S, input: S::Input) -> Result<S::Output, VerificationError>` which wraps each call in `tracing::info_span!(target: "verification", "chain_step", chain = %self.name, step = step.name())` and logs input/output.
  - Consumer usage: `let chain = ColdVerificationChain::new("config_load"); let parsed = chain.execute_step(&ParseToml, raw_bytes)?; let validated = chain.execute_step(&ValidateFields, parsed)?;`
- This keeps the trait system simple while providing tracing spans per step.

**HotVerificationChain** — async wrapper (feature-gated behind `tokio`):
- Same `execute_step` pattern but returns immediately, logging results to a fixed-size ring buffer (`VecDeque<VerificationResult>` with capacity 64).
- `#[cfg(feature = "tokio")] pub struct HotVerificationChain { ... }`
- `pub fn execute_step_async<S: VerifyStep + Send + 'static>(&self, step: S, input: S::Input) -> ()` — fires and forgets, logs result.
- `pub fn recent_results(&self) -> Vec<VerificationResult>` — reads from ring buffer.

**spawn_periodic_refetch function:**
```rust
use std::time::Duration;
use tokio::time::interval;

/// Spawns a background task that periodically calls `fetch_fn`, logging lifecycle events.
///
/// Logs:
/// - `info!("periodic_refetch started", resource = %resource_name)` on spawn
/// - `info!("periodic_refetch first_success", resource = %resource_name)` on first successful fetch
/// - `warn!("periodic_refetch failed", resource = %resource_name, error = %e, retry_count = %count)` on failure
/// - `info!("periodic_refetch self_healed", resource = %resource_name, downtime_ms = %ms)` on success after failure
/// - `error!("periodic_refetch exit", resource = %resource_name)` if task panics or is cancelled
pub fn spawn_periodic_refetch<T, E, F, Fut>(
    resource_name: String,
    interval_duration: Duration,
    fetch_fn: F,
) -> tokio::task::JoinHandle<()>
where
    T: Send + 'static,
    E: std::fmt::Display + Send + 'static,
    F: Fn() -> Fut + Send + 'static,
    Fut: std::future::Future<Output = Result<T, E>> + Send,
{
    tokio::spawn(async move {
        tracing::info!(target: "boot_resilience", resource = %resource_name, "periodic_refetch started");

        let mut ticker = interval(interval_duration);
        let mut first_success = false;
        let mut consecutive_failures: u32 = 0;
        let mut failure_started_at: Option<tokio::time::Instant> = None;

        loop {
            ticker.tick().await;

            match (fetch_fn)().await {
                Ok(_) => {
                    if !first_success {
                        first_success = true;
                        tracing::info!(target: "boot_resilience", resource = %resource_name, "periodic_refetch first_success");
                    }
                    if consecutive_failures > 0 {
                        let downtime_ms = failure_started_at
                            .map(|t| t.elapsed().as_millis())
                            .unwrap_or(0);
                        tracing::info!(
                            target: "boot_resilience",
                            resource = %resource_name,
                            downtime_ms = %downtime_ms,
                            "periodic_refetch self_healed"
                        );
                    }
                    consecutive_failures = 0;
                    failure_started_at = None;
                }
                Err(e) => {
                    if consecutive_failures == 0 {
                        failure_started_at = Some(tokio::time::Instant::now());
                    }
                    consecutive_failures += 1;
                    tracing::warn!(
                        target: "boot_resilience",
                        resource = %resource_name,
                        error = %e,
                        retry_count = %consecutive_failures,
                        "periodic_refetch failed"
                    );
                }
            }
        }
    })
}
```
