          - Posts to `channels.leaderboard`

       - Load state via `loadState()`

       - Do NOT call `.start()` — node-cron v4 auto-starts
       - Log: `logger.info('Scheduler started: 3 cron tasks active')`
       - Export: `module.exports = { startScheduler }`

       **Important:** Wrap each cron callback body in try/catch with `logger.error` — a failing cron task must not crash the bot.
