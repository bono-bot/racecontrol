**3. Handler: public_events_list (EVT-07)**
- No auth required
- Query:
```sql
SELECT id, name, description, track, car, car_class, sim_type, status,
       starts_at, ends_at, reference_time_ms, created_at,
       (SELECT COUNT(*) FROM hotlap_event_entries WHERE event_id = hotlap_events.id) as entry_count
FROM hotlap_events
WHERE status != 'cancelled'
ORDER BY
  CASE status
    WHEN 'active' THEN 1
    WHEN 'upcoming' THEN 2
    WHEN 'scoring' THEN 3
    WHEN 'completed' THEN 4
  END,
  starts_at DESC
```
- Optional status filter via WHERE clause
- Optional sim_type filter
- Return `{ "events": [...] }` with all fields + entry_count

**4. Handler: public_event_leaderboard (EVT-03, EVT-04)**
- Path param: `id`
- First fetch event metadata:
```sql
SELECT id, name, description, track, car, car_class, sim_type, status,
       starts_at, ends_at, reference_time_ms
FROM hotlap_events WHERE id = ?
```
- If not found, return `{ "error": "Event not found" }` with appropriate status
- Then fetch entries with driver display names:
```sql
SELECT hee.driver_id,
       CASE WHEN d.show_nickname_on_leaderboard = 1 AND d.nickname IS NOT NULL
            THEN d.nickname ELSE d.name END as display_name,
       hee.lap_time_ms, hee.sector1_ms, hee.sector2_ms, hee.sector3_ms,
       hee.position, hee.points, hee.badge, hee.gap_to_leader_ms,
       hee.within_107_percent, hee.result_status, hee.entered_at,
       l.car as vehicle, l.sim_type
FROM hotlap_event_entries hee
LEFT JOIN drivers d ON d.id = hee.driver_id
LEFT JOIN laps l ON l.id = hee.lap_id
WHERE hee.event_id = ?
ORDER BY hee.position ASC NULLS LAST
```
- Determine car_classes: query `SELECT DISTINCT l.car_class FROM hotlap_event_entries hee JOIN laps l ON l.id = hee.lap_id WHERE hee.event_id = ?`
- Return:
```json
{
  "event": { ...metadata },
  "car_classes": ["gt3", "gt4"],
  "entries": [ { driver_id, display_name, lap_time_ms, position, badge, gap_to_leader_ms, within_107_percent, vehicle, ... } ]
}
```

**2. Handler: public_championships_list**
- No auth
- Query:
```sql
SELECT c.id, c.name, c.description, c.season, c.car_class, c.sim_type,
       c.status, c.scoring_system, c.total_rounds, c.completed_rounds, c.created_at
FROM championships c
WHERE c.status != 'cancelled'
ORDER BY
  CASE c.status WHEN 'active' THEN 1 WHEN 'upcoming' THEN 2 WHEN 'completed' THEN 3 END,
  c.created_at DESC
```
- Return `{ "championships": [...] }`

**3. Handler: public_championship_standings (CHP-03)**
- Path param: `id`
- Fetch championship metadata first. If not found, return 404 JSON.
- Call compute_championship_standings(&state.db, &id) from Plan 04
- Also fetch per-round breakdown for each driver (round_number -> points earned):
```sql
SELECT cr.round_number, cr.event_id, he.name as event_name,
       hee.driver_id, hee.points, hee.position, hee.result_status
FROM championship_rounds cr
INNER JOIN hotlap_events he ON he.id = cr.event_id
LEFT JOIN hotlap_event_entries hee ON hee.event_id = cr.event_id
WHERE cr.championship_id = ?
ORDER BY cr.round_number, hee.position
```
- Shape response:
```json
{
  "championship": { id, name, status, car_class, ... },
  "standings": [ { position, driver_id, display_name, total_points, wins, p2_count, p3_count, rounds_entered, best_result } ],
  "rounds": [ { round_number, event_id, event_name, results: [ { driver_id, points, position, result_status } ] } ]
}
```

**4. Handler: public_event_sessions (GRP-02, GRP-03)**
- Path param: `id` (event_id)
- Query group sessions linked to this event:
```sql
SELECT gs.id, gs.status, gs.started_at, gs.completed_at
FROM group_sessions gs
WHERE gs.hotlap_event_id = ?
ORDER BY gs.started_at DESC
```
- For each session, get multiplayer results:
```sql
SELECT mr.driver_id,
       CASE WHEN d.show_nickname_on_leaderboard = 1 AND d.nickname IS NOT NULL
            THEN d.nickname ELSE d.name END as display_name,
       mr.position, mr.best_lap_ms, mr.total_time_ms, mr.laps_completed, mr.dnf
FROM multiplayer_results mr
LEFT JOIN drivers d ON d.id = mr.driver_id
WHERE mr.group_session_id = ?
ORDER BY mr.position ASC
```
- Compute F1 points per driver inline: `f1_points_for_position(position, dnf == 1)`
- Include gap_to_leader: best_lap_ms - min(best_lap_ms) among non-DNF
- Return:
```json
{
  "event_id": "...",
  "sessions": [
    {
      "session_id": "...",
      "status": "completed",
      "results": [
        { "position": 1, "driver_id": "...", "display_name": "...", "best_lap_ms": 80000, "race_points": 25, "gap_to_leader_ms": 0, "dnf": false }
      ]
    }
  ]
}
```
