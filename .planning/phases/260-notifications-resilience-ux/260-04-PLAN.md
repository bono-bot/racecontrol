  <action>
    1. In db/mod.rs run_migrations(), add:
       ```sql
       CREATE TABLE IF NOT EXISTS virtual_queue (
         id TEXT PRIMARY KEY,
         driver_id TEXT NOT NULL,
         driver_name TEXT,
         phone TEXT,
         party_size INTEGER NOT NULL DEFAULT 1,
         status TEXT NOT NULL DEFAULT 'waiting' CHECK(status IN ('waiting','called','seated','left','expired')),
         position INTEGER,
         estimated_wait_minutes INTEGER,
         joined_at TEXT DEFAULT (datetime('now')),
         called_at TEXT,
         seated_at TEXT,
         updated_at TEXT DEFAULT (datetime('now'))
       )
       CREATE INDEX IF NOT EXISTS idx_queue_status ON virtual_queue(status, joined_at)
       ```

    2. Add POST /queue/join to public_routes:
       - Accepts: { driver_name, phone, party_size (optional, default 1) }
       - Optional: driver_id (if logged in customer from PWA)
       - INSERT into virtual_queue with status='waiting'.
       - Compute position: COUNT(*) WHERE status='waiting' AND joined_at < this entry's joined_at + 1.
       - Estimate wait: position * avg_session_minutes (config or default 30min) / total_pods (8).
       - Return: { queue_id, position, estimated_wait_minutes }

    3. Add GET /queue/status/{id} to public_routes:
       - Returns current position, estimated_wait_minutes, status for the given queue entry.
       - Recompute position dynamically (count of waiting entries ahead).

    4. Add GET /queue to staff_routes:
       - Returns all entries with status='waiting' or 'called', ordered by joined_at.
       - Staff dashboard shows the full queue.

    5. Add POST /queue/{id}/call to staff_routes:
       - UPDATE status='called', called_at=now. Staff calls the next customer.

    6. Add POST /queue/{id}/seat to staff_routes:
       - UPDATE status='seated', seated_at=now. Customer is now gaming.

    7. Add POST /queue/{id}/leave to public_routes:
       - UPDATE status='left'. Customer can remove themselves from queue.

    8. Add a background cleanup: in the main server startup, spawn a task that every 5 minutes expires 'called' entries older than 10 minutes (UPDATE status='expired' WHERE status='called' AND called_at < datetime('now', '-10 minutes')).

    9. All handlers: no .unwrap(), proper error returns.
