<objective>
Fix /relay/chain/run so it returns chain results synchronously instead of timing out with 504.

Adding the missing WS handler fixes both CHAIN-10 and CHAIN-11 in a single targeted change.

Output: james/index.js with a chain_result handler added after the delegate_result handler (line ~644). Existing exec_result and delegate_result handlers are untouched.
