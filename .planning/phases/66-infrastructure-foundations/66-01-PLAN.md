<task type="auto">
  <name>Task 1: Discover NIC adapter name and set static IP on server .23 via rc-agent</name>
  <files></files>
  <action>
All commands run remotely on server .23 via rc-agent :8090 POST from James.

Step 1 — Discover the NIC adapter name:
POST http://192.168.31.23:8090/exec with:
{"cmd": "powershell -Command \"Get-NetAdapter | Where-Object {$_.Status -eq 'Up'} | Select-Object Name, InterfaceIndex, MacAddress | Format-Table -AutoSize\"", "timeout_ms": 10000}

Record the adapter name that has MAC address 10-FF-E0-80-B1-A7. It may be "Ethernet", "Ethernet 2", etc.

Step 2 — Check current IP config:
{"cmd": "powershell -Command \"Get-NetIPAddress -InterfaceAlias '<ADAPTER_NAME>' -AddressFamily IPv4 | Format-Table IPAddress, PrefixLength, PrefixOrigin\"", "timeout_ms": 10000}

If already shows 192.168.31.23 as Manual/Static, skip to Step 4.

Step 3 — Set static IP (replace <ADAPTER_NAME> with discovered name):
First remove existing IP config to avoid dual-IP:
{"cmd": "powershell -Command \"Remove-NetIPAddress -InterfaceAlias '<ADAPTER_NAME>' -Confirm:$false -ErrorAction SilentlyContinue; Remove-NetRoute -InterfaceAlias '<ADAPTER_NAME>' -Confirm:$false -ErrorAction SilentlyContinue\"", "timeout_ms": 10000}

Then set static IP:
{"cmd": "powershell -Command \"New-NetIPAddress -InterfaceAlias '<ADAPTER_NAME>' -IPAddress 192.168.31.23 -PrefixLength 24 -DefaultGateway 192.168.31.1\"", "timeout_ms": 10000}

Then disable DHCP on the interface (prevents dual-IP per Pitfall 2):
{"cmd": "powershell -Command \"Set-NetIPInterface -InterfaceAlias '<ADAPTER_NAME>' -Dhcp Disabled\"", "timeout_ms": 10000}

Then set DNS:
{"cmd": "powershell -Command \"Set-DnsClientServerAddress -InterfaceAlias '<ADAPTER_NAME>' -ServerAddresses 192.168.31.1\"", "timeout_ms": 10000}

Step 4 — Verify static IP is set:
{"cmd": "powershell -Command \"Get-NetIPAddress -InterfaceAlias '<ADAPTER_NAME>' -AddressFamily IPv4 | Format-Table IPAddress, PrefixLength, PrefixOrigin -AutoSize\"", "timeout_ms": 10000}

Expect: IPAddress 192.168.31.23, PrefixLength 24, PrefixOrigin Manual.

IMPORTANT: Write JSON payloads to a temp file and use `curl -d @file` — bash string escaping mangles backslashes in PowerShell commands (per CLAUDE.md Git Bash JSON rule).

DANGER: Never include "RCAGENT_SELF_RESTART" in any cmd string sent to rc-agent.
  </action>

  </verify>
  <done>Server .23 NIC has static IP 192.168.31.23 with gateway 192.168.31.1 and DNS 192.168.31.1. Get-NetIPAddress shows PrefixOrigin Manual.</done>
</task>
