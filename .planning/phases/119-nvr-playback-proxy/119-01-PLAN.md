Create `crates/rc-sentry-ai/src/nvr.rs` implementing a Dahua NVR CGI API client.

The Dahua HTTP CGI API for file search uses these steps:
1. `POST /cgi-bin/mediaFileFind.cgi?action=factory&object=` -- creates a search handle (returns `result=ID`)
2. `POST /cgi-bin/mediaFileFind.cgi?action=findFile&object=ID&condition.Channel=N&condition.StartTime=YYYY-MM-DD%20HH:MM:SS&condition.EndTime=YYYY-MM-DD%20HH:MM:SS&condition.Types[0]=dav&condition.Flags[0]=Event` -- starts search
3. `POST /cgi-bin/mediaFileFind.cgi?action=findNextFile&object=ID&count=100` -- gets results
4. `POST /cgi-bin/mediaFileFind.cgi?action=close&object=ID` -- closes handle
5. `GET /cgi-bin/RPC_Loadfile/path/to/file.dav` -- streams the file bytes

All requests use HTTP Digest Auth (admin/Admin@123).

Implement:

```rust
use reqwest::Client;
use serde::Serialize;

pub struct NvrClient {
    client: Client,
    base_url: String,
    username: String,
    password: String,
}

#[derive(Debug, Clone, Serialize)]
pub struct NvrFileInfo {
    pub channel: u32,
    pub start_time: String,
    pub end_time: String,
    pub file_path: String,
    pub file_size: u64,
    pub file_type: String,
}
```

Methods on NvrClient:
- `pub fn new(config: &NvrConfig) -> Self` -- creates reqwest client with digest auth support
- `pub async fn search_files(&self, channel: u32, start: &str, end: &str) -> anyhow::Result<Vec<NvrFileInfo>>` -- runs the 4-step mediaFileFind flow, parses the key=value response format
- `pub async fn stream_file(&self, file_path: &str) -> anyhow::Result<reqwest::Response>` -- returns the raw response from RPC_Loadfile for streaming

For digest auth, use reqwest's built-in support: `client.get(url).basic_auth()` won't work -- Dahua uses digest. Actually reqwest does NOT support digest auth natively. Instead, implement a simple digest auth helper:
- Make initial request, get 401 with WWW-Authenticate header
- Parse realm, nonce, qos from the header
- Compute HA1 = MD5(username:realm:password), HA2 = MD5(method:uri), response = MD5(HA1:nonce:nc:cnonce:qop:HA2)
- Retry with Authorization: Digest header

Use md-5 crate (add to Cargo.toml: `md-5 = "0.10"`) for MD5 computation.

Parse the Dahua response format which is key=value lines:
```
status.FilePath=/mnt/dvr/2026-03-22/001/dav/09/09.30.00-09.45.00[M][0@0][0].dav
status.Channel=1
status.StartTime=2026-03-22 09:30:00
status.EndTime=2026-03-22 09:45:00
status.Length=12345678
status.Type=dav
```

Add `mod nvr;` to main.rs (just the mod declaration, no wiring yet -- that's plan 02).
