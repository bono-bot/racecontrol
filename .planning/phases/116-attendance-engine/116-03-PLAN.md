   - Define shared state:
     ```rust
     pub struct AttendanceState {
         pub db_path: String,
         pub present_timeout_secs: u64,
         pub min_shift_hours: u64,
     }
     ```
   - `pub fn attendance_router(state: Arc<AttendanceState>) -> axum::Router`:
     ```rust
     Router::new()
         .route("/api/v1/attendance/present", get(present_handler))
         .route("/api/v1/attendance/history", get(history_handler))
         .route("/api/v1/attendance/shifts", get(shifts_day_handler))
         .route("/api/v1/attendance/shifts/:person_id", get(shifts_person_handler))
         .with_state(state)
     ```
   - `present_handler`: Compute `day` as today in IST (`chrono::FixedOffset::east_opt(19800)`), compute `since` as `Utc::now() - present_timeout_secs`, query `get_present_persons` via `spawn_blocking`. Return JSON: `{ "present": [...], "count": N, "timeout_secs": 1800 }`.
   - `history_handler`: Accept `Query<HistoryParams>` with `day: Option<String>` (default to today IST). Query `get_attendance_for_day` via `spawn_blocking`. Return JSON: `{ "day": "...", "entries": [...], "count": N }`.
   - `shifts_day_handler`: Accept `Query<ShiftsDayParams>` with `day: Option<String>` (default today IST). Query `get_shifts_for_day`. For each shift, add `"complete": shift_minutes.unwrap_or(0) >= min_shift_hours * 60` boolean. Return JSON: `{ "day": "...", "shifts": [...], "count": N }`.
   - `shifts_person_handler`: Accept `Path(person_id)` and optional `Query<LimitParams>` with `limit: Option<u32>` (default 30). Query `get_shifts_for_person`. Return JSON: `{ "person_id": N, "shifts": [...], "count": N }`.
   - All handlers: on DB error return 500 with `{ "error": "..." }`.
