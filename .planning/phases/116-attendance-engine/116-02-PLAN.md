1. Add to `crates/rc-sentry-ai/src/attendance/db.rs`:
   - Extend `create_tables` to also create:
     ```sql
     CREATE TABLE IF NOT EXISTS staff_shifts (
         id INTEGER PRIMARY KEY AUTOINCREMENT,
         person_id INTEGER NOT NULL,
         person_name TEXT NOT NULL,
         day TEXT NOT NULL,
         clock_in TEXT NOT NULL,
         clock_out TEXT,
         shift_minutes INTEGER,
         UNIQUE(person_id, day)
     );
     CREATE INDEX IF NOT EXISTS idx_shifts_person_day ON staff_shifts(person_id, day);
     CREATE INDEX IF NOT EXISTS idx_shifts_day ON staff_shifts(day);
     ```
   - `pub fn upsert_shift(conn: &Connection, person_id: i64, person_name: &str, day: &str, timestamp: &str) -> rusqlite::Result<ShiftAction>`:
     - Try `SELECT clock_in FROM staff_shifts WHERE person_id = ?1 AND day = ?2`.
     - If no row: INSERT with `clock_in = timestamp, clock_out = NULL, shift_minutes = NULL`. Return `ShiftAction::ClockIn`.
     - If row exists: UPDATE `clock_out = timestamp`, compute `shift_minutes` as difference between clock_in and timestamp in minutes. Return `ShiftAction::Update`.
   - `pub fn get_shift(conn: &Connection, person_id: i64, day: &str) -> rusqlite::Result<Option<ShiftEntry>>`.
   - `pub fn get_shifts_for_day(conn: &Connection, day: &str) -> rusqlite::Result<Vec<ShiftEntry>>`.
   - `pub fn get_shifts_for_person(conn: &Connection, person_id: i64, limit: u32) -> rusqlite::Result<Vec<ShiftEntry>>` -- returns last N shifts ordered by day DESC.
   - Define types:
     ```rust
     #[derive(Debug, Clone, Serialize)]
     pub struct ShiftEntry {
         pub id: i64,
         pub person_id: i64,
         pub person_name: String,
         pub day: String,
         pub clock_in: String,
         pub clock_out: Option<String>,
         pub shift_minutes: Option<i64>,
     }

     #[derive(Debug, Clone, PartialEq)]
     pub enum ShiftAction {
         ClockIn,   // First recognition of the day
         Update,    // Subsequent recognition, clock_out updated
     }
     ```
   - Add unit tests: `test_upsert_clock_in`, `test_upsert_updates_clock_out`, `test_get_shifts_for_day`, `test_shift_unique_per_person_per_day`.
   - Also call `pub fn is_staff(conn: &Connection, person_id: i64) -> rusqlite::Result<bool>` that checks `SELECT role FROM persons WHERE id = ?1` returns `'staff'`.
