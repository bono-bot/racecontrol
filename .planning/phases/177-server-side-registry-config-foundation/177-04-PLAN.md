**In `state.rs` — `broadcast_flag_sync()` method (lines 400-426):**

Replace the current implementation that builds one global `HashMap<String, bool>` and sends it to all pods. Instead:

1. Read the feature_flags cache once (keep the existing read lock).
2. Compute `version` the same way (max of all row versions).
3. In the `for (pod_id, sender)` loop over `agent_senders`, build a **per-pod** flag map:
   - For each flag row, parse `row.overrides` as `HashMap<String, bool>` using `serde_json::from_str`. If parse fails or overrides is empty `"{}"`, use `row.enabled`.
   - Check if the parsed overrides HashMap contains the current `pod_id` key. If yes, use that override value. If no, use `row.enabled`.
4. Build `FlagSyncPayload { flags: per_pod_flags, version }` and send to that specific pod.

The key change: move the flag_map construction INSIDE the sender loop so each pod gets its own resolved values.

Concrete code pattern for the resolution:
```rust
let cache = self.feature_flags.read().await;
let version = cache.values().map(|r| r.version as u64).max().unwrap_or(0);
let agent_senders = self.agent_senders.read().await;
for (pod_id, sender) in agent_senders.iter() {
    let flags: HashMap<String, bool> = cache
        .values()
        .map(|row| {
            let effective = serde_json::from_str::<HashMap<String, bool>>(&row.overrides)
                .ok()
                .and_then(|ovr| ovr.get(pod_id).copied())
                .unwrap_or(row.enabled);
            (row.name.clone(), effective)
        })
        .collect();
    let payload = FlagSyncPayload { flags, version };
    if let Err(e) = sender.send(CoreToAgentMessage::FlagSync(payload)).await {
        tracing::warn!("Failed to send FlagSync to pod {}: {}", pod_id, e);
    }
}
```

**In `ws/mod.rs` — `FlagCacheSync` handler (lines 834-841):**

Apply the same per-pod resolution. The current code builds `flag_map` using `row.enabled` for all flags. Change it to resolve overrides for the specific `payload.pod_id`:

Replace:
```rust
let flag_map: std::collections::HashMap<String, bool> = flags
    .iter()
    .map(|(name, row)| (name.clone(), row.enabled))
    .collect();
```

With:
```rust
let pod_id = &payload.pod_id;
let flag_map: std::collections::HashMap<String, bool> = flags
    .iter()
    .map(|(name, row)| {
        let effective = serde_json::from_str::<std::collections::HashMap<String, bool>>(&row.overrides)
            .ok()
            .and_then(|ovr| ovr.get(pod_id).copied())
            .unwrap_or(row.enabled);
        (name.clone(), effective)
    })
    .collect();
```

No changes to `FlagSyncPayload` struct needed — it already carries `HashMap<String, bool>`, we just resolve the correct bool per pod before populating it.

Do NOT use `.unwrap()` anywhere. Use `.ok()` and `.unwrap_or()` for safe fallback.
