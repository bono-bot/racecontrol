Add a public method `pub async fn load_feature_flags(&self)` on AppState that:
1. Queries `SELECT * FROM feature_flags` using `sqlx::query_as::<_, FeatureFlagRow>`
2. Populates `self.feature_flags` write lock with name -> row HashMap
3. Also queries `SELECT COALESCE(MAX(seq_num), 0) FROM config_push_queue` and sets `self.config_push_seq` to max + 1

Add a public method `pub async fn broadcast_flag_sync(&self)` that:
1. Reads `self.feature_flags` to build `HashMap<String, bool>` of name -> enabled
2. Gets the max version from the cache (or 0)
3. Iterates `self.agent_senders.read().await` and sends `CoreToAgentMessage::FlagSync(FlagSyncPayload { flags, version })` to each pod
4. Logs warnings on send failure (same pattern as broadcast_settings)

use axum::{extract::{State, Path, Extension}, Json};
use axum::http::StatusCode;
use serde::{Serialize, Deserialize};

#[derive(Debug, Clone, sqlx::FromRow, Serialize, Deserialize)]

    pub name: String,
    pub enabled: bool,
    pub default_value: bool,

pub struct CreateFlagRequest {
    pub name: String,
    #[serde(default)]

    #[serde(default)]
    pub default_value: bool,
    #[serde(default)]
    pub overrides: Option<serde_json::Value>,
}

#[derive(Debug, Deserialize)]
pub struct UpdateFlagRequest {
    pub enabled: Option<bool>,
    pub default_value: Option<bool>,
    pub overrides: Option<serde_json::Value>,
}

Handler functions:

**`list_flags`**: `GET /api/v1/flags`
- Read from `state.feature_flags` RwLock cache
- Return `Json(Vec<FeatureFlagRow>)` sorted by name

**`create_flag`**: `POST /api/v1/flags`
- Extract `Extension(claims): Extension<StaffClaims>`, `Json(body): Json<CreateFlagRequest>`
- Validate name is non-empty, alphanumeric + underscores, max 64 chars. Return 400 with `{"errors": {"name": "reason"}}` on failure.
- Validate overrides keys match `pod_N` pattern if provided
- Insert into `feature_flags` table with version=1, overrides default to `'{}'` if None
- Insert audit log entry: action="create", entity_type="feature_flag", entity_name=name, old_value=null, new_value=JSON of full flag, pushed_by=claims.sub
- Update the `state.feature_flags` cache
- Call `state.broadcast_flag_sync().await`
- Return 201 with the created flag row

**`update_flag`**: `PUT /api/v1/flags/:name`
- Extract `Path(name)`, `Extension(claims)`, `Json(body): Json<UpdateFlagRequest>`
- Read current flag from cache. Return 404 if not found.
- Build UPDATE SET clause for only the provided fields + increment version + set updated_at
- Execute UPDATE on DB
- Insert audit log entry: action="update", old_value=JSON of old state, new_value=JSON of new state
- Update the `state.feature_flags` cache with new row
- Call `state.broadcast_flag_sync().await`
- Return 200 with updated flag row

All DB errors mapped to 500 with `{"error": e.to_string()}`. No `.unwrap()` anywhere.
