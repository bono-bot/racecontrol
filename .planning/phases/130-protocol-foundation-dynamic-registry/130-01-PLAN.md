2. **DynamicCommandRegistry** class with:
   - Constructor: `constructor({ safeEnv })` — accepts a frozen safeEnv object (from buildSafeEnv()). Internal storage is a `new Map()`.
   - `register({ name, binary, args, tier, timeoutMs, description, cwd, allowedEnvKeys })` — validates binary against ALLOWED_BINARIES. If not allowed, throws `Error("Binary '${binary}' is not in the allowed binaries list")`. If allowed, stores the full spec object in the Map keyed by name. `allowedEnvKeys` defaults to `[]`. `timeoutMs` defaults to `30000`. Returns the stored spec.
   - `get(name)` — returns the spec object or undefined.
   - `has(name)` — returns boolean.
   - `remove(name)` — deletes from Map, returns boolean (true if existed).
   - `list()` — returns `Array.from(this.#commands.entries()).map(([name, spec]) => ({ name, description: spec.description, tier: spec.tier }))` — NEVER exposes binary or args.
   - `buildCommandEnv(name)` — looks up the command, starts with `{ ...this.#safeEnv }`. If the command has `allowedEnvKeys` array with entries, for each key, if `process.env[key]` is defined, adds it to the env object. Returns `Object.freeze(result)`. If command not found, returns `this.#safeEnv`.
   - `toJSON()` — returns `Array.from(this.#commands.entries()).map(([name, spec]) => ({ name, ...spec }))`.
   - `fromJSON(arr)` — clears the Map, then calls `this.register(item)` for each item in arr. Silently skips items whose binary is no longer in ALLOWED_BINARIES (logs a warning to console.warn).
   - `get size()` — returns `this.#commands.size`.
