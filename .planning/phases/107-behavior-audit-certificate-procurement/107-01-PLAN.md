Severity classification rules (from research):
- CRITICAL: Immediate ban trigger or near-certain detection (ReadProcessMemory on game PID, kernel drivers)
- HIGH: Known detection signature (SetWindowsHookEx, continuous process enumeration with kills during game)
- MEDIUM: Contributing risk factor (unsigned binaries, GPU contention from Ollama, overlay windows)
- LOW: Minimal risk but should be documented (named shared memory reads, UDP sockets, registry writes at startup)
- NONE: No anti-cheat risk (health HTTP endpoint, WebSocket, billing, lock screen)
