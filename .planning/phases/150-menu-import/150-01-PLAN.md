    1. **cafe.rs** — Add 3 async handler functions:

       **import_preview** (`pub async fn import_preview`):
       - Signature: `(State(state): State<Arc<AppState>>, mut multipart: Multipart) -> Result<Json<serde_json::Value>, StatusCode>`
       - Read the "file" field from multipart. Get the filename to detect format (.xlsx vs .csv).
       - If filename ends with `.xlsx` or `.xls`, call `parse_xlsx_bytes(&bytes)`. If `.csv`, call `parse_csv_bytes(&bytes)`. Otherwise return 400.
       - Run `detect_column_mapping` on the headers. Return the mapping so frontend can show/override it.
       - For each parsed row, call `validate_import_row` and build `ImportRowResult`.
       - Return JSON response matching Research Pattern 1 preview shape:
         ```json
         {
           "columns": [{"index": 0, "header": "Item Name", "mapped_to": "name"}, ...],
           "rows": [{"row_num": 1, "name": "...", "category": "...", "selling_price": "...", "cost_price": "...", "description": "...", "valid": true, "errors": []}],
           "total_rows": N,
           "valid_rows": N,
           "invalid_rows": N
         }
         ```

       **confirm_import** (`pub async fn confirm_import`):
       - Signature: `(State(state): State<Arc<AppState>>, Json(req): Json<ConfirmImportRequest>) -> Result<Json<serde_json::Value>, StatusCode>`
       - `struct ConfirmImportRequest { rows: Vec<ConfirmedImportRow> }` (Deserialize)
       - Call `confirm_import_rows(&state.db, &req.rows).await`
       - Return `{"imported": N}` on success, 500 on error with `tracing::warn!`

       **upload_item_image** (`pub async fn upload_item_image`):
       - Signature: `(State(state): State<Arc<AppState>>, Path(id): Path<String>, mut multipart: Multipart) -> Result<Json<serde_json::Value>, StatusCode>`
       - Verify item exists: `SELECT id FROM cafe_items WHERE id = ?` — return 404 if not found
       - Read "file" field from multipart
       - Decode image: `image::load_from_memory(&bytes).map_err(|_| StatusCode::BAD_REQUEST)?`
       - Resize if width > 800: `img.resize(800, u32::MAX, image::imageops::FilterType::Lanczos3)`
       - Encode to JPEG: `resized.write_to(&mut cursor, image::ImageFormat::Jpeg)`
       - Generate filename: `format!("{}.jpg", uuid::Uuid::new_v4())`
       - Create dir: `tokio::fs::create_dir_all("./data/cafe-images").await`
       - Write file: `tokio::fs::write(format!("./data/cafe-images/{}", filename), jpeg_bytes).await`
       - If item had a previous image_path, delete old file: `tokio::fs::remove_file(old_path).await` (ignore errors)
       - Update DB: `UPDATE cafe_items SET image_path = ?, updated_at = datetime('now') WHERE id = ?`
       - Return `{"image_url": "/static/cafe-images/{filename}"}`
