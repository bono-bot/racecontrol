<objective>
Eliminate all silent failures in rc-agent config loading, racecontrol config loading, process guard empty allowlist handling, and rc-sentry FSM transitions. After this plan, every degraded state emits an observable signal at the moment it occurs.

Purpose: Three major incident classes (silent config fallback, empty allowlist mass violations, unlogged FSM transitions) each caused multi-hour outages because no signal was emitted when the system entered a degraded state. This plan adds warn!/error!/RecoveryLogger entries at every silent failure point.

Output: Modified rc-agent/main.rs, racecontrol/config.rs, rc-agent/process_guard.rs, rc-sentry/watchdog.rs with observable state transition logging.
</objective>
