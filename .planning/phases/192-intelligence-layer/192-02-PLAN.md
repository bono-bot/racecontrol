Create `audit/lib/delta.sh` with one exported function, following core.sh style (comment headers, `export -f`):

**Function: `compute_delta`**
- Takes no args. Uses `$RESULT_DIR` (current run) and calls `find_previous_run` for the previous run dir.
- If no previous run exists (`find_previous_run` returns empty), write `{"has_previous": false, "entries": []}` to `$RESULT_DIR/delta.json` and return 0.
- Otherwise:

**Step 1 — Collect current results:**
```bash
# Merge all phase-*.json in RESULT_DIR into a single array
jq -s '.' "$RESULT_DIR"/phase-*.json > "$tmp_current"
```

**Step 2 — Collect previous results:**
```bash
local prev_dir
prev_dir=$(find_previous_run)
jq -s '.' "$prev_dir"/phase-*.json > "$tmp_previous"
```

**Step 3 — jq delta join on composite key (phase + host):**
Use a single jq expression that:
1. Creates a lookup map from previous results keyed by `(.phase + "|" + .host)`
2. For each current result, looks up the previous entry by the same key
3. Assigns a category based on status transition:

| Previous Status | Current Status | Category |
|----------------|---------------|----------|
| (not found) | any | NEW_ISSUE (if FAIL/WARN) or NEW_STABLE (if PASS/QUIET) |
| PASS | FAIL | REGRESSION |
| PASS | WARN | REGRESSION |
| FAIL | PASS | IMPROVEMENT |
| WARN | PASS | IMPROVEMENT |
| FAIL | FAIL | PERSISTENT |
| PASS | PASS | STABLE |
| PASS | QUIET | STABLE (NOT regression — mode/venue aware) |
| QUIET | PASS | STABLE (venue reopened — not an improvement) |
| QUIET | FAIL | REGRESSION (was quiet, now actively failing) |
| QUIET | QUIET | STABLE |
| WARN | WARN | PERSISTENT |
| WARN | FAIL | REGRESSION |
| FAIL | WARN | IMPROVEMENT |
| any | QUIET where venue_state changed | STABLE (venue context change) |

4. Also checks for entries in the previous run that are NOT in the current run (phases that were removed/skipped).

The jq expression (write as a heredoc to a temp file for readability):
```bash
local jq_script
jq_script=$(cat <<'JQ'
def categorize(prev; curr):
  if prev == null then
    if (curr.status == "FAIL" or curr.status == "WARN") then "NEW_ISSUE"
    else "NEW_STABLE" end
  elif prev.venue_state != curr.venue_state then
    # Venue state changed — context shift, not a real transition
    if curr.status == "FAIL" and prev.status != "FAIL" then "REGRESSION"
    elif curr.status == "PASS" and prev.status == "FAIL" then "IMPROVEMENT"
    else "STABLE" end
  elif prev.status == "PASS" and (curr.status == "FAIL" or curr.status == "WARN") then "REGRESSION"
  elif prev.status == "WARN" and curr.status == "FAIL" then "REGRESSION"
  elif (prev.status == "FAIL" or prev.status == "WARN") and curr.status == "PASS" then "IMPROVEMENT"
  elif prev.status == "FAIL" and curr.status == "WARN" then "IMPROVEMENT"
  elif prev.status == curr.status then
    if curr.status == "FAIL" then "PERSISTENT"
    elif curr.status == "WARN" then "PERSISTENT"
    else "STABLE" end
  elif prev.status == "PASS" and curr.status == "QUIET" then "STABLE"
  elif prev.status == "QUIET" and curr.status == "PASS" then "STABLE"
  elif prev.status == "QUIET" and curr.status == "FAIL" then "REGRESSION"
  elif prev.status == "QUIET" and curr.status == "QUIET" then "STABLE"
  else "STABLE" end;

($prev | map({key: (.phase + "|" + .host), value: .}) | from_entries) as $prev_map |
($curr | map({
  phase: .phase,
  host: .host,
  tier: .tier,
  prev_status: ($prev_map[.phase + "|" + .host].status // null),
  curr_status: .status,
  prev_venue_state: ($prev_map[.phase + "|" + .host].venue_state // null),
  curr_venue_state: .venue_state,
  severity: .severity,
  message: .message,
  category: categorize($prev_map[.phase + "|" + .host]; .)
})) as $entries |
{
  has_previous: true,
  previous_dir: $prev_dir,
  counts: {
    regression: [$entries[] | select(.category == "REGRESSION")] | length,
    improvement: [$entries[] | select(.category == "IMPROVEMENT")] | length,
    persistent: [$entries[] | select(.category == "PERSISTENT")] | length,
    new_issue: [$entries[] | select(.category == "NEW_ISSUE")] | length,
    stable: ([$entries[] | select(.category == "STABLE" or .category == "NEW_STABLE")] | length)
  },
  entries: $entries
}
JQ
)
```

Call jq with:
```bash
jq -n --slurpfile prev "$tmp_previous" --slurpfile curr "$tmp_current" --arg prev_dir "$prev_dir" \
  '$prev[0] as $prev | $curr[0] as $curr | '"$jq_script" > "$RESULT_DIR/delta.json"
```
