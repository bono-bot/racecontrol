  <behavior>
    - SessionEnforcer::new(sim_type: SimType, pid: u32, duration_secs: u64) -> Self
      - Stores launch time, duration, pid, warned flag
    - SessionEnforcer::tick() -> SessionAction { Continue, Warn(remaining_secs), Terminate }
      - At duration - 60s: return Warn(60) (one-time)
      - At duration: return Terminate
      - Otherwise: return Continue
    - SessionEnforcer::terminate(pid: u32) -> Result<(), String>
      - taskkill /F /PID {pid} via hidden_cmd (same pattern as clean_state_reset)
    - ProcessMonitor::new(pid: u32, sim_type: SimType) -> Self
    - ProcessMonitor::check() -> ProcessStatus { Running, Exited(Option<i32>) }
      - Uses is_process_alive(pid). If not alive, return Exited with exit code if available.
    - ProcessMonitor applies to ALL non-AC sims (F1, iRacing, LMU, Forza, FH5, AC EVO, AC Rally)
    - SessionEnforcer only applies to ForzaHorizon5 (and potentially Forza Motorsport)
  </behavior>
  <action>
    Create `crates/rc-agent/src/session_enforcer.rs` with two structs:

    1. **SessionEnforcer** (GAME-03): For open-world games (ForzaHorizon5) that have no session end event.
       - `new(sim_type, pid, duration_secs)` — stores Instant::now() as start, duration, pid
       - `tick(&mut self) -> SessionAction` — compares elapsed vs duration. Returns Warn(remaining) once at T-60s, Terminate at T+0
       - `terminate(pid: u32) -> Result<(), String>` — calls `taskkill /F /PID {pid}` via hidden_cmd. Uses CREATE_NO_WINDOW on Windows.
       - Warning delivery: the caller (event_loop) will send a WS message to server which triggers the kiosk overlay. Enforcer just signals.

    2. **ProcessMonitor** (GAME-08): Generic crash detection for non-AC sims.
       - `new(pid, sim_type)` — stores pid and sim_type
       - `check(&self) -> ProcessStatus` — calls `game_process::is_process_alive(self.pid)`. Returns Running or Exited(exit_code).
       - Exit code extraction: on Windows, use `OpenProcess + GetExitCodeProcess` via winapi if available, otherwise None.
       - This is the fallback crash detection for games without shared memory telemetry (everything except AC original).

    Both structs are `Send + Sync` compatible (no raw pointers, just primitives).

    SessionAction enum: `Continue`, `Warn { remaining_secs: u64 }`, `Terminate`.
    ProcessStatus enum: `Running`, `Exited { exit_code: Option<i32> }`.

    Register `pub mod session_enforcer;` in lib.rs.

    Use `#[cfg(test)]` guards on actual process-killing functions. Tests verify tick() timing logic with synthetic Instant values.
  </action>
