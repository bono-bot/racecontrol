  <behavior>
    - Test: encrypt_field("hello") then decrypt_field(result) == "hello" (roundtrip)
    - Test: encrypt_field("hello") called twice produces DIFFERENT ciphertext (random nonce)
    - Test: decrypt_field("not-base64") returns Err
    - Test: decrypt_field(base64 of only 5 bytes) returns Err (too short for nonce)
    - Test: hash_phone("9876543210") == hash_phone("9876543210") (deterministic)
    - Test: hash_phone("+919876543210") == hash_phone("9876543210") (strips +91 prefix)
    - Test: hash_phone("9876543210") != hash_phone("1234567890") (different phones different hashes)
    - Test: load_encryption_keys() returns Err when env vars unset
    - Test: load_encryption_keys() returns Ok(FieldCipher) when both env vars set to valid 64-char hex
    - Test: load_encryption_keys() returns Err when RACECONTROL_ENCRYPTION_KEY is not 64 hex chars
  </behavior>
  <action>
    1. Add workspace dependencies to root Cargo.toml [workspace.dependencies]:
       aes-gcm = "0.10"
       hmac = "0.12"
       sha2 = "0.10"
       hex = "0.4"

    2. Add to crates/racecontrol/Cargo.toml [dependencies]:
       aes-gcm = { workspace = true }
       hmac = { workspace = true }
       sha2 = { workspace = true }
       hex = { workspace = true }

    3. Create crates/racecontrol/src/crypto/mod.rs:
       pub mod encryption;

    4. Create crates/racecontrol/src/crypto/encryption.rs with:
       - FieldCipher struct holding Aes256Gcm cipher and hmac_key: Vec<u8>
       - FieldCipher::new(aes_key: &[u8; 32], hmac_key: &[u8]) -> Self
       - FieldCipher::encrypt_field(&self, plaintext: &str) -> Result<String, String>
         Returns base64(nonce || ciphertext). Uses OsRng for random 12-byte nonce.
       - FieldCipher::decrypt_field(&self, encoded: &str) -> Result<String, String>
         Decodes base64, splits nonce (first 12 bytes) from ciphertext, decrypts.
       - FieldCipher::hash_phone(&self, phone: &str) -> String
         Normalizes phone (trim, strip "+91" prefix), HMAC-SHA256 with key, returns hex string.
       - pub fn load_encryption_keys() -> Result<FieldCipher, String>
         Reads RACECONTROL_ENCRYPTION_KEY and RACECONTROL_HMAC_KEY from env vars.
         Both must be valid hex. Encryption key must decode to exactly 32 bytes.
         Returns descriptive error messages on failure.

       Use these imports:
       use aes_gcm::{aead::{Aead, KeyInit, OsRng}, Aes256Gcm, Nonce, AeadCore};
       use hmac::{Hmac, Mac};
       use sha2::Sha256;
       use base64::{engine::general_purpose::STANDARD as BASE64, Engine};

    5. Add `pub mod crypto;` to lib.rs (alphabetically near existing mod declarations).

    6. Write #[cfg(test)] mod tests in encryption.rs with all behaviors listed above.
       For tests that call load_encryption_keys(), use std::env::set_var/remove_var
       inside a serial test (since env vars are process-global, run with --test-threads=1).
  </action>
