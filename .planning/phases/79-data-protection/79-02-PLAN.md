       - For each row (batch of 100 in a transaction):

         - email_enc = if email is not empty, cipher.encrypt_field(email)
         - name_enc = if name is not empty, cipher.encrypt_field(name)
         - guardian_phone_hash = if guardian_phone is not empty, cipher.hash_phone(guardian_phone)
         - guardian_phone_enc = if guardian_phone is not empty, cipher.encrypt_field(guardian_phone)
         - UPDATE drivers SET phone_hash=?, phone_enc=?, email_enc=?, name_enc=?, guardian_phone_hash=?, guardian_phone_enc=?, phone=NULL, email=NULL, guardian_phone=NULL WHERE id=?
       - Keep name column populated (needed for leaderboard until nickname adoption). Only NULL phone, email, guardian_phone.
       - Log count of migrated rows at INFO level (no PII in log).
       - This function is idempotent: rows with phone_hash already set are skipped.
