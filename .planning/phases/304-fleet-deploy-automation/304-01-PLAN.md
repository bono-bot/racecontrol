```rust
#[derive(Deserialize)]
pub struct FleetDeployRequest {
    pub binary_hash: String,        // Expected SHA256 of binary
    pub binary_url: String,         // HTTP URL to download from
    pub scope: DeployScope,         // all | canary | pods([u32])
    #[serde(default = "default_wave_delay")]
    pub wave_delay_secs: u64,       // Default 5
    #[serde(default)]
    pub force: bool,                // Override weekend peak lock
}
fn default_wave_delay() -> u64 { 5 }

#[derive(Deserialize, Serialize, Clone, Debug)]
#[serde(rename_all = "snake_case")]
pub enum DeployScope { All, Canary, Pods(Vec<u32>) }

#[derive(Serialize, Clone, Debug)]
pub struct FleetDeploySession {
    pub deploy_id: String,
    pub binary_hash: String,
    pub binary_url: String,
    pub scope: DeployScope,
    pub wave_delay_secs: u64,
    pub initiated_by: String,
    pub initiated_at: String,       // RFC3339 IST
    pub current_wave: u8,           // 0=not started, 1=canary, 2, 3
    pub overall_status: DeployOverallStatus,
    pub waves: Vec<WaveStatus>,
    pub rollback_events: Vec<RollbackEvent>,
}

#[derive(Serialize, Clone, Debug, PartialEq)]
#[serde(rename_all = "snake_case")]
pub enum DeployOverallStatus { Pending, Running, Completed, Failed, RollingBack }

#[derive(Serialize, Clone, Debug)]
pub struct WaveStatus {
    pub wave_number: u8,
    pub pods: Vec<String>,
    pub status: WaveDeployStatus,
    pub started_at: Option<String>,
    pub completed_at: Option<String>,
    pub pod_results: Vec<PodDeployResult>,
}

#[derive(Serialize, Clone, Debug, PartialEq)]
#[serde(rename_all = "snake_case")]
pub enum WaveDeployStatus { Pending, Running, Passed, Failed }

#[derive(Serialize, Clone, Debug)]
pub struct PodDeployResult {
    pub pod_id: String,
    pub status: String,   // "complete" | "failed" | "waiting_session" | "rolled_back" | "skipped"
    pub detail: Option<String>,
}

#[derive(Serialize, Clone, Debug)]
pub struct RollbackEvent {
    pub wave: u8,
    pub pod_id: String,
    pub reason: String,
    pub rolled_back_at: String,
    pub outcome: String,  // "success" | "failed"
}
```

**2. Helper: `create_session()`** — builds a FleetDeploySession from the request, generating deploy_id as `format!("{}-{}", &binary_hash[..8], chrono::Utc::now().timestamp())`. Pre-populates `waves` based on scope:
- `All` → 3 waves using WAVE_1, WAVE_2, WAVE_3 constants
- `Canary` → 1 wave with just WAVE_1
- `Pods(ids)` → 1 wave with the specified pods

**3. Core: `pub async fn run_fleet_deploy(state: Arc<AppState>, session_lock: Arc<RwLock<Option<FleetDeploySession>>>)`**

The function reads the session from the lock, then orchestrates:

a. **Resolve pod IPs** from `state.pods` — filter to only pods in scope. If a pod in scope is not connected (not in `state.pods`), mark it as `skipped` in results.

b. **Set OTA sentinel + kill switch** on all target pods: `set_ota_sentinel()` and `set_kill_switch(active=true)`.

c. **For each wave (1, 2, 3):**
   - Update session: `current_wave = wave_num`, wave status = Running, `started_at = now()`. Use the snapshot-drop pattern:
     ```rust
     {
         let mut guard = session_lock.write().await;
         if let Some(ref mut s) = *guard {
             s.current_wave = wave_num;
             s.waves[wave_idx].status = WaveDeployStatus::Running;
             s.waves[wave_idx].started_at = Some(now_ist_rfc3339());
         }
     } // guard dropped before .await
     ```
   - For each pod in wave:
     - **Billing check:** `{ let timers = state.billing.active_timers.read().await; timers.contains_key(&pod_id) }`. If active session: set `WaitingSession` in `pod_deploy_states`, store `binary_url` in `pending_deploys`, record `PodDeployResult { status: "waiting_session" }`, continue to next pod.
     - **Deploy:** `crate::deploy::deploy_pod(state.clone(), pod_id.clone(), pod_ip.clone(), binary_url.clone()).await`
     - **Read state immediately:** `{ let states = state.pod_deploy_states.read().await; states.get(&pod_id).cloned() }` — check for `Complete` or `Failed`.
     - **If Complete:** record `PodDeployResult { status: "complete" }`.
     - **If Failed:** 
       - For **canary (wave 1):** call `rollback_wave()` for just this pod. Record `RollbackEvent`. Set `overall_status = Failed`. Clear OTA sentinel + kill switch. Return early.
       - For **non-canary (wave 2/3):** call `rollback_wave()` for just this pod. Record `RollbackEvent` and `PodDeployResult { status: "rolled_back" }`. Continue to next pod (non-fatal).
   - Mark wave as `Passed` (or `Failed` if canary failed — but we already returned).
   - **Inter-wave delay:** `tokio::time::sleep(Duration::from_secs(wave_delay_secs)).await` between waves.

d. **Cleanup:** `clear_ota_sentinel()` and `set_kill_switch(active=false)`. Set `overall_status = Completed`.

**4. Helper: `now_ist_rfc3339()`** — returns current time in IST as RFC3339 string using `chrono::Utc::now() + chrono::Duration::hours(5) + chrono::Duration::minutes(30)` with `FixedOffset::east_opt(5*3600+30*60)`.
