PART B: Create audit/audit.sh

CRITICAL: do NOT put set -e anywhere in this file.

Structure:
1. Shebang: #!/usr/bin/env bash
2. Comment block (3 lines): filename, usage, exit code meanings
3. Shell options: set -u then set -o pipefail (no set -e)
4. Resolve SCRIPT_DIR using BASH_SOURCE[0] with cd and pwd
5. Source SCRIPT_DIR/lib/core.sh, exit 2 on failure
6. Constants (define then export): SERVER_IP=192.168.31.23, SERVER_PORT=8080,
   SERVER_OPS_PORT=8090, AUTH_ENDPOINT, FLEET_HEALTH_ENDPOINT, PODS (8 IPs, space-separated),
   DEFAULT_TIMEOUT=10
7. Defaults: AUDIT_MODE="" AUDIT_TIER="" AUDIT_PHASE="" AUTO_FIX=false NOTIFY=false COMMIT=false DRY_RUN=false
8. Argument parsing (while case loop over ):
   - --mode: validate against quick|standard|full|pre-ship|post-incident, exit 2 if invalid
   - --tier, --phase: read next arg
   - --auto-fix, --notify, --commit, --dry-run: set to true
   - --help/-h: print usage and exit 0
   - unknown flag: print ERROR and exit 2
   After loop: if AUDIT_MODE is empty, print ERROR --mode is required and exit 2
   Export all flag variables.
9. Function check_prerequisites: 3 guards each exiting 2:
   - jq not found: print ERROR: jq is required. Install: winget install jqlang.jq
   - curl not found: print ERROR: curl is required
   - AUDIT_PIN empty: print ERROR: AUDIT_PIN env var is required (export AUDIT_PIN=261121)
10. Function init_result_dir:
    - IST timestamp via TZ=Asia/Kolkata date '+%Y-%m-%d_%H-%M'
    - RESULT_DIR=/results/, export RESULT_DIR, mkdir -p
    - Write run-meta.json with jq -n (fields: mode, started_at, dry_run, pin_provided)
    - Create latest symlink, fallback to latest.txt if symlink fails
11. Function acquire_auth:
    - SESSION_TOKEN=, export SESSION_TOKEN
    - Warn to stderr if empty (do NOT exit)
    - For full mode: start background subshell refreshing token every 840 seconds
12. Main body:
    - Call check_prerequisites, init_result_dir, acquire_auth
    - Echo start message with mode and result_dir
    - If DRY_RUN: echo DRY RUN and exit 0
    - VENUE_STATE=, export VENUE_STATE
    - Call emit_result with args: "00" "0" "james-local" "PASS" "P3" "Audit framework initialized"  
13. Exit code: count phase-*.json files in RESULT_DIR where .status=="FAIL" via jq.
    If FAIL_COUNT > 0: exit 1. Else: exit 0.
