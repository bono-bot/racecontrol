Create `scripts/fix_log.sh` — a bash script that interactively collects 5 structured fields and appends them to LOGBOOK.md.

**Script requirements:**

1. **Shebang and setup:**
   ```
   #!/usr/bin/env bash
   # fix_log.sh — Cause Elimination Process (GATE-05)
   # Prompts for 5 structured debugging fields and appends to LOGBOOK.md.
   # Standing rule: any bug taking >30 min to isolate MUST use this process.
   set -euo pipefail
   ```

2. **Resolve LOGBOOK.md path:**
   ```
   REPO_ROOT="$(cd "$(dirname "$0")/.." && pwd)"
   LOGBOOK="$REPO_ROOT/LOGBOOK.md"
   ```
   Verify LOGBOOK.md exists, exit 1 with error if not.

3. **Collect timestamp:**
   Generate IST timestamp: `date -u -d '+5 hours 30 minutes' '+%Y-%m-%d %H:%M IST'` (or fallback for systems without GNU date: use `TZ=Asia/Kolkata date '+%Y-%m-%d %H:%M IST'`).

4. **Prompt for each field interactively (read from stdin):**

   For each field, print a prompt with description, then read multiline input until the user types a blank line (or EOF). Each field is REQUIRED — if the input is empty (after trimming whitespace), print `"ERROR: <field_name> is required. Cannot skip."` and re-prompt.

   Fields in order:
   - **Symptom:** `"1/5 — SYMPTOM: What exactly happened? (error message, behavior observed, screenshot path)"` — read until blank line
   - **Hypotheses:** `"2/5 — HYPOTHESES: List ALL possible causes (one per line, include software/hardware/config/network/user error):"` — read until blank line. Must have at least 2 hypotheses (enforce: count lines, if < 2: `"ERROR: List at least 2 hypotheses. Single-hypothesis debugging skips the elimination step."`)
   - **Elimination:** `"3/5 — ELIMINATION: For each hypothesis, what test was run and what was the result? (one per line, format: 'H1: tested X — result Y — ELIMINATED/CONFIRMED'):"` — read until blank line
   - **Confirmed cause:** `"4/5 — CONFIRMED CAUSE: Which hypothesis survived elimination?"` — single line, required
   - **Verification:** `"5/5 — VERIFICATION: How was the fix confirmed to work? (command run, output observed, screenshot path):"` — read until blank line

5. **Format and append to LOGBOOK.md:**

   Append the following markdown block:
   ```
   ### Cause Elimination — <timestamp>

   **Symptom:** <symptom text>

   **Hypotheses:**
   <hypotheses text, each line prefixed with "- ">

   **Elimination:**
   <elimination text, each line prefixed with "- ">

   **Confirmed cause:** <cause text>

   **Verification:** <verification text>

   ---
   ```

   Append AFTER the last `---` separator in LOGBOOK.md (or at the end if no separator found). Use `echo >> "$LOGBOOK"` for appending.

6. **Confirmation output:**
   After appending, print:
   ```
   "Entry appended to LOGBOOK.md at <timestamp>"
   "File: <absolute path to LOGBOOK.md>"
   ```

7. **Make executable:** The file should have `chmod +x` (the Write tool creates it, executor should `chmod +x scripts/fix_log.sh` after).
