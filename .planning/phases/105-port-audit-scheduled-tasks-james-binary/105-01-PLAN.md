1. `pub(crate) fn parse_netstat_listening(stdout: &str) -> Vec<(u16, u32)>`
   - Split each line on whitespace
   - Skip lines where parts[0] != "TCP" (case-insensitive)
   - Skip lines where parts[3] != "LISTENING" (case-insensitive check: parts.get(3).map_or(false, |s| s.eq_ignore_ascii_case("LISTENING")))
   - Parse local address from parts[1]: split on ':' and take last segment as u16 port
   - Parse PID from parts[4] as u32
   - Return Vec<(port, pid)>

2. `pub(crate) async fn run_port_audit(whitelist: &Arc<RwLock<MachineWhitelist>>, tx: &mpsc::Sender<AgentMessage>, machine_id: &str)`
   - Read whitelist.ports and violation_action under a brief read lock, then drop lock
   - shell-out in spawn_blocking: Command::new("netstat").args(["-ano"]) with #[cfg(windows)] creation_flags(0x08000000)
   - collect stdout String from output.stdout
   - call parse_netstat_listening(&stdout) -> Vec<(port, pid)>
   - for each (port, pid): if port is in allowed_ports, skip
   - NOT in allowed: log_guard_event("PORT_VIOLATION port={} pid={}", port, pid)
   - if violation_action == "kill_and_report": attempt kill — use kill_process_verified with sysinfo to get start_time for that PID; if PID not found in sysinfo, call taskkill /F /PID {pid} directly in spawn_blocking (fallback)
   - send AgentMessage::ProcessViolation with violation_type: ViolationType::Port, name: port.to_string(), exe_path: None, action_taken: "killed"/"reported", consecutive_count: 1
