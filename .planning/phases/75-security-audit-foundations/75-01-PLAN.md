**Section 3: CORS, HTTPS, and Auth State (AUDIT-05)**

3a. **CORS** — Document the exact configuration from main.rs:
- `AllowOrigin::predicate`: matches `http://localhost:*`, `http://127.0.0.1:*`, `http://192.168.31.*`, `http://kiosk.rp*`, `*racingpoint.cloud*`
- `allow_methods`: GET, POST, PUT, PATCH, DELETE, OPTIONS

3c. **Auth infrastructure state**:
