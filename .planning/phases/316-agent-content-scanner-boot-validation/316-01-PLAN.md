<objective>
Extend content_scanner.rs to scan Steam library paths via libraryfolders.vdf parsing and probe non-Steam game exe paths, then build a GameInventory and expose functions to send it over WS (GameInventoryUpdate) and to start a 5-minute periodic rescan loop.

Purpose: The server currently has no visibility into which games are installed on each pod. The existing scanner only covers AC content. This plan gives every pod a full game inventory that the server will use for combo validation (Plan 02) and the Reliability Dashboard (Phase 319).

Output: Extended content_scanner.rs with four new pub functions — scan_steam_library, scan_non_steam_games, build_game_inventory, send_game_inventory_update — plus a spawn_inventory_rescan_loop async fn for 5-min periodic rescan.
</objective>

**parse_vdf_library_paths(vdf_path: &Path) -> Vec<PathBuf>**
- Read `libraryfolders.vdf` with `std::fs::read_to_string`. On error, log WARN and return empty vec.
- Parse line by line with `std::str::lines()`. Look for lines matching the pattern `"path"  "..."` (the key `path` and a quoted string value). Extract the string value.
- Also include the default Steam install dir `C:\Program Files (x86)\Steam` always (it may not appear in libraryfolders.vdf for the primary library).
- Dedup the list. Return `Vec<PathBuf>` where each path points to the Steam root (not `steamapps/`).
- No external crate needed — pure string parsing only.

    (2488620, SimType::F125,             "f1_25",                "F1 25"),

**scan_steam_library(pod_id: &str) -> Vec<InstalledGame>**
- Default VDF path: `C:\Program Files (x86)\Steam\steamapps\libraryfolders.vdf`
- Call `parse_vdf_library_paths` to get all library root paths.
- For each library root, check `{root}\steamapps\appmanifest_{app_id}.acf` for each entry in STEAM_APP_IDS.
- When a manifest is found:
  - Probe the exe by resolving the install dir from the appmanifest. Read the acf file, look for `"installdir"  "..."` line. Construct exe path as `{root}\steamapps\common\{installdir}\{known_exe}`.
  - Known exe per sim_type (use first match): AssettoCorsa→`acs.exe`, F125→`F1_25.exe`, IRacing→`iRacingUI.exe`.
  - `launchable = exe_path.is_file()`.
  - `scan_method = "steam_library"`.
  - `scanned_at = Utc::now().to_rfc3339()` (import `chrono::Utc`).
  - Dedup by `game_id` — if the same game_id appears in multiple libraries, keep only the first (primary library wins).
- Log INFO: `"Steam library scan: {} games found across {} library paths"`.
- Return `Vec<InstalledGame>`. Never panic — all I/O wrapped in `if let Ok(...)` or `.ok()`.

**Constraint:** No `.unwrap()` anywhere. All filesystem errors log WARN and continue (fail-open). This function is called from `spawn_blocking` — it performs blocking I/O but must not hold any async lock.
