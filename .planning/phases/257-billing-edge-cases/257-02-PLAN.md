  <action>
    1. Create `crates/rc-agent/src/inactivity_monitor.rs`:
       ```rust
       pub struct InactivityMonitor {
           threshold_secs: u64,
           last_input_at: Instant,
           alert_sent: bool,
       }
       ```
       - `new(threshold_secs: u64)` — default 600 (10 min), configurable via agent config.
       - `record_input()` — resets `last_input_at` to now, clears `alert_sent`.
       - `tick() -> Option<InactivityAlert>` — if elapsed > threshold && !alert_sent, set alert_sent=true and return Some. Otherwise None.
       - `reset()` — for session start/end, clears all state.

    2. In `crates/rc-common/src/protocol.rs`, add to `AgentMessage` enum:
       ```rust
       InactivityAlert { pod_id: String, idle_seconds: u64 }
       ```

    3. In `crates/rc-agent/src/event_loop.rs`:
       - Add `inactivity_monitor: Option<InactivityMonitor>` to `ConnectionState`.
       - When a game launches (GameStateUpdate::Running), create `InactivityMonitor::new(600)`.
       - When game ends (GameStateUpdate::Idle/Exit), set to None.
       - In the 1-second tick loop, call `inactivity_monitor.tick()`. If Some(alert), send `AgentMessage::InactivityAlert` to server via WS.
       - On any telemetry data received (steering angle, pedal position, button press), call `inactivity_monitor.record_input()`. The telemetry receivers already exist in event_loop — add a call at the point where UDP telemetry is processed.

    4. On the server side in `billing.rs`, handle `AgentMessage::InactivityAlert`:
       - Broadcast `DashboardEvent::InactivityAlert { pod_id, idle_seconds, driver_name, session_id }` to staff dashboard.
       - Log at WARN level: `"BILL-01: Pod {} idle for {}s — staff alerted"`.
       - Do NOT auto-end the session. Staff decides.

    5. Add `pub mod inactivity_monitor;` to `crates/rc-agent/src/lib.rs` (or main.rs, wherever modules are declared).
  </action>

    3. In `lock_screen.rs`, update the `show_active_session_screen()` HTML/JS:
       - The JS countdown timer already exists. Enhance it:
