1. **SafeMode struct** (exact fields from CONTEXT.md decisions):
   ```rust
   pub struct SafeMode {
       pub active: bool,
       pub game: Option<SimType>,
       pub cooldown_until: Option<std::time::Instant>,
   }
   ```
   Methods: `new()`, `enter(&mut self, game: SimType)`, `start_cooldown(&mut self) -> Instant`, `exit(&mut self)`.
   - `enter()` sets active=true, game=Some(game), cooldown_until=None (clears any pending cooldown)
   - `start_cooldown()` sets cooldown_until=Some(Instant::now() + 30s), game=None, keeps active=true
   - `exit()` sets all to false/None
   - All transitions log via `tracing::info!(target: "safe-mode", ...)`

2. **is_protected_game(sim: SimType) -> bool**: Returns true for F125, IRacing, LeMansUltimate, AssettoCorsaEvo. Returns false for AssettoCorsa, Forza, ForzaHorizon5, AssettoCorsaRally.

3. **PROTECTED_EXE_NAMES: &[&str]**: `["F1_25.exe", "iRacingSim64DX11.exe", "Le Mans Ultimate.exe", "AssettoCorsaEVO.exe", "AC2-Win64-Shipping.exe", "WRC.exe"]`

4. **exe_to_sim_type(exe_name: &str) -> Option<SimType>**: Case-insensitive mapping. F1_25.exe -> F125, iRacingSim64DX11.exe -> IRacing, "Le Mans Ultimate.exe" -> LeMansUltimate, AssettoCorsaEVO.exe/AC2-Win64-Shipping.exe -> AssettoCorsaEvo, WRC.exe -> None (no SimType variant yet).

5. **spawn_wmi_watcher() -> std::sync::mpsc::Receiver<String>**: Spawns `std::thread::spawn` running PowerShell `Register-WmiEvent` for `Win32_ProcessStartTrace`. Uses CREATE_NO_WINDOW (0x08000000). Filters output lines against PROTECTED_EXE_NAMES. Sends matching exe names over `std::sync::mpsc::channel`. On PowerShell spawn failure, logs error and returns the receiver (sender dropped = receiver returns Err on try_recv, safe no-op). Follow the hidden_cmd pattern from ai_debugger.rs. The PowerShell script:
   ```powershell
   $names = @('F1_25.exe','iRacingSim64DX11.exe','Le Mans Ultimate.exe','AssettoCorsaEVO.exe','AC2-Win64-Shipping.exe','WRC.exe')
   $query = "SELECT * FROM Win32_ProcessStartTrace"
   Register-WmiEvent -Query $query -SourceIdentifier 'RCSafeModeWatch' | Out-Null
   while ($true) {
       $event = Wait-Event -SourceIdentifier 'RCSafeModeWatch' -Timeout 5
       if ($event -ne $null) {
           $exe = $event.SourceEventArgs.NewEvent.ProcessName
           if ($names -contains $exe) { Write-Output $exe; [Console]::Out.Flush() }
           Remove-Event -SourceIdentifier 'RCSafeModeWatch'
       }
   }
   ```

6. **detect_running_protected_game() -> Option<SimType>**: One-time sysinfo scan (same pattern as game_process.rs). Iterates processes, checks against PROTECTED_EXE_NAMES via case-insensitive match, returns first match via exe_to_sim_type(). Guard with `#[cfg(not(test))]` for the sysinfo call — in tests, return None.

7. **Unit tests** in `#[cfg(test)] mod tests` block covering all behaviors listed above. Tests must NOT spawn real PowerShell or real sysinfo scans. Test SafeMode state transitions as pure struct manipulation. Test is_protected_game and exe_to_sim_type as pure functions.
