**Phase 19 fix (SF-01) -- audit/phases/tier3/phase19.sh:**

The current code (lines 32-38) checks pod health endpoint and if status=ok, hardcodes `horiz="1920"; vert="1080"` with comment "Assume minimum". This defeats the resolution check entirely.

Replace lines 32-38 with actual resolution measurement via rc-agent exec:

1. Use safe_remote_exec to query resolution. Preferred approach (simpler quoting for cmd.exe):
   ```bash
   response=$(safe_remote_exec "$ip" "8090" \
     'powershell -Command "(Get-CimInstance Win32_VideoController | Select-Object -First 1).CurrentHorizontalResolution.ToString() + [char]120 + (Get-CimInstance Win32_VideoController | Select-Object -First 1).CurrentVerticalResolution.ToString()"' \
     15)
   ```
   Note: [char]120 is 'x' -- avoids quoting issues with literal 'x' in string concatenation.

   If Get-CimInstance approach has issues, fallback to:
   ```bash
   response=$(safe_remote_exec "$ip" "8090" \
     'wmic path Win32_VideoController get CurrentHorizontalResolution,CurrentVerticalResolution /format:csv' \
     15)
   ```
   Note: wmic may not exist on Win11 26200 (comment on line 30 says so). In that case stick with Get-CimInstance.

2. Parse the response:
   ```bash
   local res_output; res_output=$(printf '%s' "$response" | jq -r '.stdout // ""' 2>/dev/null | tr -d '[:space:]')
   horiz=$(printf '%s' "$res_output" | grep -oE '^[0-9]+')
   vert=$(printf '%s' "$res_output" | grep -oE '[0-9]+$')
   ```

3. If safe_remote_exec fails (empty horiz/vert), try health endpoint as fallback:
   ```bash
   if [[ -z "$horiz" || -z "$vert" ]]; then
     local pod_health; pod_health=$(http_get "http://${ip}:8090/health" "$DEFAULT_TIMEOUT")
     local pod_up; pod_up=$(printf '%s' "$pod_health" | jq -r '.status // ""' 2>/dev/null)
     if [[ "$pod_up" = "ok" ]]; then
       status="WARN"; severity="P2"
       message="Pod healthy but resolution query failed -- verify display manually"
       emit_result "$phase" "$tier" "${host}-resolution" "$status" "$severity" "$message" "$mode" "$venue_state"
       pod_num=$((pod_num+1)); continue
     fi
     # Pod completely offline -- handled by existing venue_state logic below
     horiz="0"; vert="0"
   fi
   ```

4. Keep all the existing resolution comparison logic (7680x1440 = Surround, >=1920x1080 = single, 1024x768 = collapsed, etc.) -- the if/elif chain starting around line 40.

5. Remove the old hardcoded comment "Assume minimum; physical check for NVIDIA Surround".
