**Function: bat_validate_syntax(file_path, label)**
Validates a bat file for 5 known anti-patterns. Uses grep/sed on the file content (passed as a path to a temp file). Returns violations as lines of text.

Checks (each with line number and suggested fix):
1. **UTF-8 BOM**: `xxd -l 3 "$file_path" | grep -q 'efbb bf'` -- if found, report "Line 1: UTF-8 BOM detected — remove first 3 bytes (sed '1s/^\xef\xbb\xbf//' or re-create without BOM)"
2. **Parentheses in if/else**: `grep -nP '^\s*if\s.*\(' "$file_path"` or `grep -nP '^\s*\)\s*else\s*\(' "$file_path"` -- report "Line N: parentheses in if/else block — use goto labels instead (standing rule)"
3. **/dev/null redirection**: `grep -n '/dev/null' "$file_path"` -- report "Line N: /dev/null is Unix — use NUL or >nul on Windows"
4. **timeout command**: `grep -nP '^\s*timeout\s' "$file_path"` -- report "Line N: timeout command fails in non-interactive context — use 'ping -n N 127.0.0.1 >nul'"
5. **taskkill without restart**: For each `taskkill` line, check if there is a corresponding `start` or `schtasks` later in the file for the same process. This is a heuristic: extract the process name from `/IM <name>` and search for `start.*<name>` or `schtasks.*<name>` within 20 lines after. If not found, warn "Line N: taskkill /IM <name> without matching restart — process will stay dead"

Return value: number of violations found. Print each violation to stdout.

**Function: bat_scan_pod(pod_num, bat_name, canonical_path)**
Scans a single pod for a specific bat file.
1. Compute pod IP via `pod_ip "$pod_num"`
2. Build the remote path: `C:\\RacingPoint\\${bat_name}` (double-escaped for JSON)
3. Write JSON payload to temp file: `{"path":"C:\\\\RacingPoint\\\\${bat_name}"}` using printf to a temp file (standing rule: never inline JSON in bash)
4. Fetch via: `curl -s --max-time 10 -X POST "http://${ip}:${SENTRY_PORT}/files" -H "Content-Type: application/json" -d @"$tmpfile"`
5. If curl fails or empty response, report "POD $pod_num: UNREACHABLE" and return 1
6. Save response to temp file `$TMPDIR/pod${pod_num}_${bat_name}`
7. Compute SHA256 of canonical file (stripping \r for comparison): `tr -d '\r' < "$canonical_path" | sha256sum | awk '{print $1}'`
8. Compute SHA256 of fetched file (also stripping \r): `tr -d '\r' < "$fetched_path" | sha256sum | awk '{print $1}'`
9. If hashes match: print "POD $pod_num: MATCH ($bat_name)"
10. If hashes differ: print "POD $pod_num: DRIFT ($bat_name)" then show diff output: `diff --color=auto <(tr -d '\r' < "$canonical_path") <(tr -d '\r' < "$fetched_path")` with labels `--label "canonical" --label "pod-$pod_num"`
11. Also run `bat_validate_syntax` on the fetched file
12. Return: 0 for match, 1 for drift or unreachable

**Function: bat_scan_all()**
Loop over pods 1-8, call `bat_scan_pod` for both `start-rcagent.bat` and `start-rcsentry.bat` (if canonical exists). Track total matches, drifts, and unreachable counts. Print summary at end.
