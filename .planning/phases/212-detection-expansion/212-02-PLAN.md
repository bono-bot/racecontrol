   ```bash
   local server_flags
   server_flags=$(curl -s --max-time 10 "${SERVER_URL}/api/v1/flags" 2>/dev/null || echo "[]")
   local canonical_enabled
   canonical_enabled=$(printf '%s' "$server_flags" | \
     jq -r '[.[] | select(.enabled==true) | .name] | sort | join(",")' 2>/dev/null || echo "")
   ```
3. If server_flags is empty or canonical_enabled is empty, log "server unreachable or no flags" and return (no baseline)

   - Query rc-agent flags: `curl -s --max-time 10 "http://${pod_ip}:8090/api/v1/flags" 2>/dev/null`
   - If empty response: `empty_count=$((empty_count + 1)); continue`
   - Extract enabled flag names sorted+joined with comma
   - Compare with canonical_enabled
   - If different: compute missing and extra flags using comm:
     ```bash
     local missing; missing=$(comm -23 \
       <(printf '%s\n' "${canonical_enabled//,/$'\n'}" | sort) \
       <(printf '%s\n' "${pod_enabled//,/$'\n'}" | sort) | tr '\n' ',' | sed 's/,$//')
     local extra; extra=$(comm -13 \
       <(printf '%s\n' "${canonical_enabled//,/$'\n'}" | sort) \
       <(printf '%s\n' "${pod_enabled//,/$'\n'}" | sort) | tr '\n' ',' | sed 's/,$//')
     ```
   - Emit P2 finding: "flag desync on ${pod_ip}: missing=[${missing}] extra=[${extra}] vs server canonical"
6. After loop: if empty_count == 8 and canonical_enabled is not empty, emit P2 finding for fleet: "all pods returned empty for /api/v1/flags -- endpoint may not exist on rc-agent"
7. Increment DETECTOR_FINDINGS for each desync

**DET-06: scripts/detectors/detect-schema-gap.sh**

Create detect_schema_gap() function:
1. `set -u; set -o pipefail` (no set -e)
2. Define critical tables and late-added columns to check (known ALTER TABLE additions from db/mod.rs):
   ```bash
   local -a SCHEMA_CHECKS=(
     "drivers:updated_at"
     "drivers:membership_type"
     "billing_sessions:payment_method"
     "billing_sessions:staff_discount"
     "feature_flags:description"
     "game_catalog:category"
   )
   ```
3. For each check in SCHEMA_CHECKS, split on `:` into table and column
4. Check venue DB (server .23 via :8090 exec):
   ```bash
   local venue_result
   venue_result=$(safe_remote_exec "192.168.31.23" 8090 \
     "sqlite3.exe C:\\RacingPoint\\racecontrol.db \"SELECT ${column} FROM ${table} LIMIT 1\"" 10)
   local venue_stderr; venue_stderr=$(printf '%s' "$venue_result" | jq -r '.stderr // ""' 2>/dev/null)
   local venue_exit; venue_exit=$(printf '%s' "$venue_result" | jq -r '.exitCode // 0' 2>/dev/null)
   local venue_has_col=true
   if printf '%s' "$venue_stderr" | grep -qi "no such column"; then venue_has_col=false; fi
   ```
5. Check cloud DB (Bono VPS via SSH -- relay custom_command not supported per STATE.md):
   ```bash
   local cloud_stderr=""
   local cloud_has_col=true
   cloud_stderr=$(ssh -o StrictHostKeyChecking=no -o ConnectTimeout=10 -o BatchMode=yes \
     root@100.70.177.44 \
     "sqlite3 /root/racecontrol/racecontrol.db 'SELECT ${column} FROM ${table} LIMIT 1'" 2>&1 || echo "ERROR")
   if printf '%s' "$cloud_stderr" | grep -qi "no such column"; then cloud_has_col=false; fi
   if printf '%s' "$cloud_stderr" | grep -qi "unable to open\|Connection refused\|Connection timed out"; then
     cloud_has_col="unknown"
   fi
   ```
6. Compare results:
   - If venue_has_col=true and cloud_has_col=false: emit P2 "schema gap: cloud DB missing ${table}.${column} (venue has it)"
   - If venue_has_col=false and cloud_has_col=true: emit P2 "schema gap: venue DB missing ${table}.${column} (cloud has it)"
   - If venue_has_col=false and cloud_has_col=false: emit P2 "schema gap: both DBs missing ${table}.${column} -- migration not applied"
   - If either is "unknown": skip (unreachable, not a schema gap)
7. Increment DETECTOR_FINDINGS for each gap
8. `export -f detect_schema_gap`
