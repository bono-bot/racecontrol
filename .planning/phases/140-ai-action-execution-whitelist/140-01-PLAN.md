       - RestartRcAgent
       - KillGame
       - ClearTemp

    3. Implement parse_ai_action(response: &str) -> Option<AiSafeAction>:
       - Search response for a JSON object containing "action" key. The LLM embeds it in prose so scan for `{` and try serde_json::from_str on substrings. A simple approach: find the first `{` and last `}` pair that parses as ActionBlock.
       - Match action field string to enum variant (exact lowercase match):
         "kill_edge" → KillEdge, "relaunch_lock_screen" → RelaunchLockScreen,
         "restart_rcagent" → RestartRcAgent, "kill_game" → KillGame, "clear_temp" → ClearTemp
       - Any other string → return None (whitelist rejection)
       - If no parseable JSON block → return None
       - No unwrap() — all parse errors return None

       Append to the prompt string:

ONLY use actions from this whitelist — any other action will be ignored:\n  kill_edge — kills all msedge.exe processes\n  relaunch_lock_screen — closes and relaunches Edge kiosk\n  restart_rcagent — restarts the rc-agent service\n  kill_game — kills the current game process\n  clear_temp — clears Windows temp files\nFormat: {"action":"<name>"}\nIf no action applies, omit the JSON block entirely.
