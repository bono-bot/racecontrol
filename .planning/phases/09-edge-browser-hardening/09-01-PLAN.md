Create a batch script `edge-harden.bat` at `C:/Users/bono/racingpoint/deploy-staging/edge-harden.bat` with these commands:

```bat
@echo off
REM Edge Browser Hardening - Racing Point Pods
REM Disables auto-update services and kiosk-breaking features

REM 1. Stop and disable EdgeUpdate service (EDGE-01)
sc stop EdgeUpdate >nul 2>&1
sc config EdgeUpdate start= disabled

REM 2. Stop and disable edgeupdate service (lowercase variant) (EDGE-01)
sc stop edgeupdate >nul 2>&1
sc config edgeupdate start= disabled

REM 3. Stop and disable MicrosoftEdgeElevationService if it exists (EDGE-01)
sc stop MicrosoftEdgeElevationService >nul 2>&1
sc config MicrosoftEdgeElevationService start= disabled 2>nul

REM 4. Create Edge policy key if it doesn't exist, set StartupBoostEnabled=0 (EDGE-02)
reg add "HKLM\SOFTWARE\Policies\Microsoft\Edge" /v StartupBoostEnabled /t REG_DWORD /d 0 /f

REM 5. Set BackgroundModeEnabled=0 (EDGE-03)
reg add "HKLM\SOFTWARE\Policies\Microsoft\Edge" /v BackgroundModeEnabled /t REG_DWORD /d 0 /f

echo EDGE_HARDEN_COMPLETE
```
