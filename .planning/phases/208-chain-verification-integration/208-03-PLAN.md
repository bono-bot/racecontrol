**Change 1 — StepValidateCriticalFields::run() (lines 595-608):**

Replace the current implementation that uses eprintln + Ok(input) with one that returns Err(TransformError) when fallbacks are detected. The config must be serialized into the raw_value so the caller can recover it:

```rust
fn run(&self, input: Config) -> Result<Config, VerificationError> {
    let default = Config::default_config();
    let mut fallbacks = Vec::new();
    if input.database.path == default.database.path {
        fallbacks.push("database.path");
    }
    if !fallbacks.is_empty() {
        // COV-03: Emit TransformError through chain for tracing span capture.
        // Config is still usable — caller catches this as non-fatal warning.
        // Using eprintln as well because tracing may not be initialized during config load.
        eprintln!("[config_validate] fields at default values: {:?}", fallbacks);
        return Err(VerificationError::TransformError {
            step: self.name().to_string(),
            raw_value: format!("fields_at_default={:?}", fallbacks),
        });
    }
    Ok(input)
}
```

After the Step 2 PID liveness check fails (line 561-565), instead of just logging and continuing, retry the spawn once. The retry should:

1. Log that a retry is being attempted
2. Re-run the spawn (session1 or schtasks — use the same method that succeeded originally)
3. Re-run PID liveness check
4. Continue to health poll regardless

Replace lines 556-577 (Step 2 + Step 3) with:

```rust

    tracing::warn!(target: LOG_TARGET,
        "COV-05: PID liveness failed for {} — retrying spawn once (method={})",
        cfg.process_name, spawn_method_name
    );

    // Retry spawn once using the same method that originally succeeded
    let retry_ok = if spawn_method_name == "session1" {
        match session1_spawn::spawn_with_start_script(&cfg.start_script) {
            Ok(_) => true,
            Err(e) => {
                tracing::error!(target: LOG_TARGET, "COV-05: retry spawn via session1 failed: {}", e);

            }

        }

            Err(e) => {
                tracing::error!(target: LOG_TARGET, error = %e, "COV-05: PID liveness still failed after retry — proceeding to health poll");

} else {
```
