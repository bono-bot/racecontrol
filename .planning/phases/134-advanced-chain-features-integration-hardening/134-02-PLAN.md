    **1. Extend ChainOrchestrator with pause/resume capability:**

    Add a `#paused` flag and an `#activeState` object that tracks the running chain's progress.

    - `#activeState` is set at the start of `execute()`: `{ chainId, steps, completedSteps, stepIndex: 0, prevStdout: '', chainTimeoutMs }`. It is updated after each step completes in `#runSteps()`.
    - `#paused` flag: when set to true, `#runSteps()` checks it at the top of each loop iteration and breaks out early (returning 'PAUSED' status).
    - `pause()` method: sets `#paused = true`. The currently awaited broker.waitFor() will eventually resolve or timeout; the loop will break before the NEXT step. Returns the current `#activeState` snapshot (deep copy).
    - `resume(savedState)` method: accepts a saved state object, sets `#paused = false`, and calls a modified `#runSteps()` that starts from `savedState.stepIndex` with `savedState.prevStdout` and `savedState.completedSteps`. Returns the full chain result (same shape as execute()).
    - `getState()` method: returns a deep copy of `#activeState` if a chain is running, null otherwise.
    - After execute() completes (any status), clear `#activeState` to null.

    **2. Create a shared introspection helper** (inline function in each index.js, or a small helper -- use judgment). The function merges COMMAND_REGISTRY entries with dynamicRegistry.list():
    ```javascript
    function buildIntrospectionResponse(queryId) {
      // Static commands: extract name, description, tier, timeoutMs (never binary/args/cwd)
      const staticCmds = Object.entries(COMMAND_REGISTRY).map(([name, spec]) => ({
        name,
        description: spec.description,
        tier: spec.tier,
        timeoutMs: spec.timeoutMs,
      }));
      // Dynamic commands: dynamicRegistry.list() already returns { name, description, tier }
      // Add timeoutMs from the full spec via dynamicRegistry.get()
      const dynamicCmds = dynamicRegistry.list().map(cmd => ({
        ...cmd,
        timeoutMs: dynamicRegistry.get(cmd.name)?.timeoutMs ?? 30000,
      }));
      return { queryId, commands: [...staticCmds, ...dynamicCmds] };
    }
    ```
