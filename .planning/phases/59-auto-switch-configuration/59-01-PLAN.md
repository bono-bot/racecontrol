<objective>
Implement the auto-switch configuration self-healing function that ensures ConspitLink can auto-detect games and load correct FFB presets on every rc-agent startup.

Purpose: ConspitLink reads Global.json from `C:\RacingPoint\Global.json` at runtime, but this file only exists in the install directory on pods. This breaks AresAutoChangeConfig silently. This plan places the file at the correct path, ensures `AresAutoChangeConfig` is `"open"`, verifies GameToBaseConfig.json mappings, and wires into the startup sequence.

Output: Working `ensure_auto_switch_config()` in ffb_controller.rs with unit tests, wired into main.rs startup before enforce_safe_state.
</objective>
