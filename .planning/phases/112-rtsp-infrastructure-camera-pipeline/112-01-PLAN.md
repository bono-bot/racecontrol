1. Create `C:\RacingPoint\start-go2rtc.bat` using bash heredoc + `sed 's/$/\r/'` (per CLAUDE.md bat file rules — never use Write tool directly for bat files to avoid UTF-8 BOM).

   Content:
   ```bat
   @echo off
   title go2rtc RTSP Relay
   cd /D C:\RacingPoint\go2rtc
   echo [%date% %time%] Starting go2rtc RTSP relay...
   go2rtc.exe
   echo [%date% %time%] go2rtc exited, restarting in 5 seconds...
   timeout /t 5 /nobreak >nul
   goto :eof
   ```

   Note: This is a simple launch script. For auto-restart on crash, use Windows Scheduled Task with restart-on-failure (not a bat loop) per research recommendation.

2. Register as HKLM Run key for auto-start on boot:
   ```
   reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" /v "go2rtc" /t REG_SZ /d "C:\RacingPoint\start-go2rtc.bat" /f
   ```

3. Verify the Run key was created:
