       - Path param: id (String)
       - Query 1 (driver stats): SELECT display_name, total_laps, total_time_ms, avatar_url, created_at FROM drivers WHERE id = ? — explicitly NO email/phone/wallet

       - Query 3 (lap history): SELECT track, car, lap_time_ms, sector1_ms, sector2_ms, sector3_ms, valid, created_at FROM laps WHERE driver_id = ? AND (suspect IS NULL OR suspect = 0) ORDER BY created_at DESC LIMIT 100
