Output: `billing_fsm.rs` module with transition table, validation function, and unified end-session function. All billing status mutations in `billing.rs` guarded by the transition table.

       - PausedManual + End -> Completed
       - PausedManual + Cancel -> Cancelled

       - This replaces the current end_billing_session() function body — call authoritative_end_session from end_billing_session
